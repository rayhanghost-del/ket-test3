## 韩国电商业务

### Coupang LCL 拼箱服务（2025-11-28版）

**服务概述：**
- **全称**：Coupang Direct LCL（Less than Container Load）拼柜服务
- **核心价值**：比行业平均成本低，配送时间缩短 2-3 天
- **服务模式**：卖家只需在 Wing 系统创建入库请求、贴标、送货至中国仓库，其余跨境运输和清关由 Coupang 处理

**两条服务线路：**

| 线路 | 仓库地址 | 建议送货时间 | 船期 | 时效 | 费率 |
|-----|---------|-------------|------|------|------|
| **威海线** | 威海市环翠区温泉镇悦丰物流园4号库 | 周日、二、四 | 周一、三、五发船 | 约6天 | **70$/CBM** |
| **深圳线** | 深圳市宝安区福永段107国道现代物流园7栋 | 周四、六、一 | 周一、三、五发船（经威海） | 约8天 | **98$/CBM** |

**目的地火箭仓：** INC14、ECH1、ANS4（可能有变化）

**报价包含：**
- 中国集货仓操作费
- 深圳至威海陆运费（仅深圳线）
- 订舱服务费、海运费及全部杂费
- 韩国进口清关服务费
- 韩国港口至 Coupang FC 陆运费
- Coupang 补贴：浮动费用、卸货费

**报价不包含（卖家承担）：**
- 国内运输到 Coupang 仓库的费用
- 进出口关税/税费
- 增值服务：保险、报关费、查验费、退税申报
- 重新贴标、换箱、包装修复、缝制产地标等

**关键时效节点（威海线示例）：**
- 入库订单确认 → 货物入库 → 出口报关 → 开船 → 到港 → 进口报关 → 缴税 → 清关放行 → FC入库
- 理想情况：6天；延迟情况：10-12天

**入库要求：**
- 单箱重 ≤30kg，长宽高之和 ≤2500mm，单边 ≤1100mm
- 单箱单品、单箱单SKU、单箱多规格单SKU
- 必须贴：产地标（MADE IN CHINA）、Coupang条形码、韩文标签（特定产品）、入仓外箱单
- 商品必须有保护措施（气泡膜等）

**锂电池特殊规定（188特殊规定）：**
- 锂金属电池：锂含量 ≤1g；电池组 ≤2g
- 锂离子电池：≤20Wh；电池组 ≤100Wh
- 需提供：MSDS认证、非危险品运输鉴定报告、UN38.3测试报告

**常见问题：**
- 同一批货物分到不同火箭仓：需分开准备出运资料
- 无需打托，纸箱形式入仓
- 快递送仓无需打印送货单（建议顺丰）

## 项目文档：KET自适应测评系统

### 项目概览
- **项目名称**：KET自适应测评系统
- **所属业务**：英语培训班
- **用途**：剑桥KET考试15分钟自适应测评
- **在线地址**：https://eng-assessment.vercel.app
- **GitHub仓库**：rayhanghost-del/ket-assessment

### 核心功能
- **自适应逻辑**：根据Part 1答题表现动态分流
  - ≥83%正确率 → Part 2高难度（音乐家主题）
  - 50-83% → Part 2中等难度（大学生主题）
  - <50% → 跳过Part 2直接到听力
- **随机抽题**：从18道题中随机抽取6道
- **隐藏得分**：避免影响后续答题心态
- **50分制评分**：A2(≥40分) / A1(25-39分) / 入门(<25分)

### 技术栈
- **前端**：纯HTML + CSS + JavaScript（无框架依赖）
- **部署**：Vercel + GitHub
- **题库格式**：JSON
- **自适应引擎**：JavaScript逻辑控制
