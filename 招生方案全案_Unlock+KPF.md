# 📋 海报设计清单

## 必备元素
- [ ] 主标题（最大字号）：Unlock + KPF
- [ ] 核心数据：12年、99%通过率
- [ ] 目标人群：小学至初三
- [ ] 行动号召：扫码预约/免费测评
- [ ] 二维码（至少5×5cm）

## 可选元素
- [ ] 老师照片（增加信任）
- [ ] 学员成绩单截图（证明成果）
- [ ] 限时/限额信息（制造紧迫感）
- [ ] 联系方式（电话/地址）

## 排版原则
1. **F型阅读** - 重要信息放左上
2. **数字突出** - 99%、12年用最大字号
3. **颜色克制** - 不超过3种主色
4. **留白充足** - 不要塞太满
5. **二维码要大** - 方便扫码
