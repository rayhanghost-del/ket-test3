# Coupang 运营深度学习资源汇总

## 一、链接合并/拆分（상품 결합/분리）

### 核心问题
- **노출상품 ID 불일치**：同一商品的不同选项显示在不同页面
- **의도하지 않은 결합**：商品被错误合并到其他卖家商品
- **분리 필요**：不同商品被错误合并在一起

### 解决方法

#### 1. 链接合并（결합）
**适用场景：**
- 同一商品的多个选项显示在不同页面
- 需要合并到同一个 상품 ID 下

**操作步骤：**
1. 쿠팡윙 → 상품관리 → 상품조회/수정
2. 확인 상품의 노출상품 ID
3. 판매자지원 → 온라인문의
4. 상품관리 → 상품수정요청 → 동일한 상품 결합
5. 대량 결합 시 엑셀 파일 다운로드 후 신청

**处理时间：** 최대 8일

#### 2. 链接拆分（분리）
**适用场景：**
- 不同商品被错误合并
- 需要分离成独立商品

**操作方式：**
- 同样通过 온라인문의 申请
- 提供 상품코드 和 분리理由

### 避免合并的方法
- 상품명差异化
- 카테고리准确选择
- 옵션设置清晰区分
- 定期监控 노출상품 ID

### 促成合并的时机
- 同一商品多个选项分散时
- 需要统一管理和库存时
- 提升搜索权重时

## 二、广告设置与 ROAS 优化

### 广告类型对比

| 广告类型 | 特点 | 适用人群 | 控制程度 |
|---------|------|---------|---------|
| **AI 스마트광고** | AI自动选择商品和优化 | 新手、没时间管理 | 低 |
| **매출 최적화** | 自选商品，AI优化关键词 | 想控制商品但不想管关键词 | 中 |
| **수동 성과형** | 自选商品、关键词、出价 | 有经验、追求效率 | 高 |
| **브랜드 인지도** | CPM模式，品牌曝光 | 品牌推广 | 中 |

### ROAS 设置策略

#### 初始设置（新手）
```
일예산: 3~5만원
목표 ROAS: 300% 이하
```

#### 优化阶段
```
성과 좋은 상품: ROAS 500%~1000%
성과一般的: ROAS 300%~500%
신규 테스트: ROAS 200%~300%
```

#### ROAS 1000% 是否可能？
**答案：可能，但有条件**

**实现条件：**
1. **高毛利商品**（毛利率 50%+）
2. **成熟商品**（有销量、好评、点击率）
3. **精准关键词**（高转化长尾词）
4. **优化后的详情页**（高转化率）

**风险提示：**
- ROAS 设置过高 → 广告曝光机会减少
- ROAS 设置过低 → 可能亏损
- 建议逐步提升，不要一步到位

### 广告优化技巧

#### 1. 商品效率指数优化
- 提升 리뷰수（评论数）
- 保持高 평점（评分）
- 增加 판매량（销量）
- 优化 클릭률（点击率）

#### 2. 关键词策略
- **初期**：使用 스마트 타겟팅（自动匹配）
- **中期**：分析보고서，添加高转化词
- **后期**：수동 입력精准关键词

#### 3. 出价策略
- **제안 입찰가**仅供参考，不一定要按建议
- **수동 광고**：搜索区出价 > 非搜索区
- **비검색 영역**：如果效果不好，降低到100원或关闭

#### 4. 广告报告分析
**重点关注指标：**
- 클릭률 (CTR) < 2% → 优化图片/标题/价格
- 전환율 (CVR) < 1% → 优化详情页/评论
- 비검색 영역 효율低 → 转브랜드 광고或수동 설정

## 六、常见问题 FAQ

### Q1: ROAS 设置 1000% 为什么没曝光？
**A:** ROAS 设置过高会导致广告机会减少。建议：
- 初始设置 300%
- 根据实际表现逐步提升
- 确保商品有高转化率基础

### Q2: 链接合并后多久生效？
**A:** 최대 8일，通常 3-5 日。建议：
- 提前规划，避免紧急需求
- 合并前备份商品信息
- 合并后检查所有选项是否正常显示

### Q3: AI 스마트광고 和 매출 최적화 哪个好？
**A:** 
- 新手/没时间：AI 스마트광고
- 想控制商品：매출 최적화
- 追求效率：最终都要转到 수동 성과형

### Q4: 广告费怎么设置最合理？
**A:** 
- 最小 일예산: 10,000원
- 推荐 초기: 30,000원
- 根据 ROAS 调整：ROAS > 3 可增加预算
