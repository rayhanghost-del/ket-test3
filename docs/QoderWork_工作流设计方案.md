# QoderWork 工作流设计方案

## 工作流1：外教简历自动整理

### 适用场景
收到大量外教简历后，自动按国籍/学科/经验年限分类整理

### 操作步骤
1. **打开QoderWork**，输入指令：
   ```
   帮我整理文件夹"外教简历"中的所有PDF和Word文件
   按以下规则分类：
   - 按国籍分到不同子文件夹（USA/UK/SA/Canada等）
   - 文件名格式：国籍_姓名_学科.pdf
   - 生成一个汇总Excel，包含姓名、国籍、学科、教龄、当前位置
   ```

2. **确认执行计划**，点击「确认执行」

3. **等待完成**，检查分类结果

### 预期效果
- 100份简历整理时间：人工2小时 → QoderWork 5分钟
- 自动生成候选人汇总表，方便后续筛选

## 工作流2：Coupang销售数据日报生成

### 适用场景
每天需要分析Coupang后台销售数据，生成运营日报

### 操作步骤
1. **从Coupang导出数据**到本地Excel

2. **打开QoderWork**，输入指令：
   ```
   分析这个Coupang销售数据表：
   - 计算今日销售额、订单量、客单价
   - 找出Top5畅销商品
   - 识别库存低于10的商品
   - 生成可视化图表（趋势图、饼图）
   - 输出一份运营日报Word文档，包含以上所有内容
   ```

3. **拖拽Excel文件**到对话框或指定路径

4. **确认执行**，等待生成报告

### 预期效果
- 数据分析+报告生成：人工1小时 → QoderWork 3分钟
- 自动生成图表和文字分析

## 工作流3：KET测评成绩批量分析

### 适用场景
英语培训班批量测评后，需要生成学员成绩分析报告

### 操作步骤
1. **准备测评数据Excel**（包含学员姓名、各部分得分）

2. **打开QoderWork**，输入指令：
   ```
   分析这份KET测评成绩表：
   - 计算每个学员的总分和等级（A1/A2/B1）
   - 统计班级平均分、最高分、最低分
   - 找出各部分的薄弱点（听力/阅读/写作/口语）
   - 为每个学员生成个性化评语
   - 制作一份家长版成绩报告PPT
   ```

3. **确认执行**，等待完成

### 预期效果
- 成绩分析+报告制作：人工3小时 → QoderWork 10分钟
- 自动生成个性化评语，无需手动撰写

## 工作流5：文件批量重命名（通用）

### 适用场景
下载的文件名混乱，需要统一格式

### 操作步骤
1. **打开QoderWork**，输入指令：
   ```
   将文件夹"下载"中的所有文件按以下规则重命名：
   - 图片：日期_序号.jpg
   - 文档：日期_原文件名.pdf
   - 删除文件名中的特殊符号和广告文字
   ```

2. **确认执行**，等待完成

### 预期效果
- 100个文件重命名：人工30分钟 → QoderWork 1分钟

## 💡 使用建议

1. **从简单任务开始**：先尝试文件整理、批量重命名
2. **明确指令**：描述越清晰，结果越准确
3. **善用沙盒**：处理敏感数据时开启沙盒模式
4. **关注Credits消耗**：标准档足够应对大部分任务
5. **保存成功指令**：好用的指令可以重复使用
