## 📍 第一站：英孚教育（兴庆南路学大大厦2层）

### 进门先看（2分钟）
- [ ] 门头是否亮灯、整洁
- [ ] 大厅家长/孩子多不多
- [ ] 展示墙有什么（学员作品/获奖/师资）
- [ ] 教室大概几间、设备如何

### 核心问题（10分钟）

#### 1. 课程
| 问题 | 对方回答 | 关键点 |
|------|----------|--------|
| "三年级有什么课程？" | | 课程名称、内容 |
| "一周几次课？一次多久？" | | 计算月课时量 |
| "一个班几个孩子？" | | 班型大小 |
| "有外教吗？哪国的？" | | 欧美/菲教/中教 |

#### 2. 价格 ⭐⭐⭐
| 问题 | 对方回答 | 记录 |
|------|----------|------|
| "学费怎么收？" | | 年费/学期费 |
| "一学期多少钱？" | ______元 | |
| "平均一节课多少钱？" | ______元 | |
| "现在报名有优惠吗？" | | 折扣/赠课 |
| "退费政策是怎样的？" | | 能否退/比例/时限 |
| "除了学费还有其他费用吗？" | | 教材/注册费 |

#### 3. 师资
| 问题 | 回答 |
|------|------|
| "任课老师什么背景？" | |

#### 4. 侧面打听竞品
| 问题 | 对方回答（记下原话） |
