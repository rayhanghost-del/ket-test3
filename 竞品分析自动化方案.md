# 竞品分析自动化脚本方案

## 目标
实现从飞书表格读取产品链接和图片，自动分析后回填数据

## 技术架构

### 方案1：纯飞书API方案（推荐）
- 读取飞书表格中的产品链接
- 调用爬虫获取Coupang页面数据
- OCR识别详情页图片
- 自动分析并回填表格

### 方案2：半自动化工具
- 你提供Excel/CSV文件（链接+图片路径）
- 脚本批量处理
- 生成回填数据

## 需要确认的问题

1. **Coupang反爬虫** - 直接爬取可能被拦截，需要
   - 使用浏览器自动化（Playwright/Selenium）
   - 或者通过你提供的酷铺助手数据

2. **图片OCR识别** - 详情页长图的文字提取
   - 使用PaddleOCR/Tesseract
   - 或调用云端OCR API

3. **AI分析部分** - 提取关键信息后生成分析文案
   - 可以复用我现在的分析逻辑
   - 封装成独立模块

## 实现步骤

### 第一阶段：基础数据自动录入（1-2天）
- 读取表格中的链接
- 自动获取：商品名、价格、评分、评论数
- 回填基础数据

### 第二阶段：详情页分析自动化（2-3天）
- 下载详情页图片
- OCR提取文字
- AI分析生成：优势特点、建议、埋词、设计建议、风格特点
- 回填分析数据

### 第三阶段：全流程自动化（1天）
- 整合流程
- 定时任务/手动触发
- 错误处理和日志

## 需要你配合的

1. 飞书表格的API权限（app_token、table_id）
2. 测试数据（5-10条产品链接）
3. 确认字段映射（表格列名对应关系）

## 预计时间
- 完整开发：3-5天
- 可用版本（第一阶段）：1-2天
