# FCE备考课程规划（家长沟通版）

## 学生背景
- **学生**：六年级男生
- **当前水平**：PET优秀通过
- **目标考试**：2025年7月中旬FCE考试
- **备考周期**：约3-4个月

---

# 📋 方案A：稳步进阶方案

## 适合家庭
- 希望稳扎稳打，不追求快节奏
- 孩子课业较忙，需要平衡校内学习
- 希望充分巩固基础，追求高分通过（Grade A/B）

## 课程概览

| 项目 | 详情 |
|-----|------|
| **总时长** | 16周（约4个月） |
| **上课频率** | 每周1次 |
| **单次时长** | 2小时 |
| **总课时** | 32小时 |
| **上课形式** | 一对一 |

## 三阶段规划

### 第一阶段：基础夯实（第1-6周）
**目标**：熟悉FCE考试结构，建立词汇语法基础，适应PET→FCE的难度跨越

**核心内容**：
- FCE考试4部分全面介绍（阅读、语用、听力、写作、口语）
- 综合教程Unit 1-6精讲
- 核心词汇系统积累（每周1个Unit，共6个Unit）
- 阅读Part 1-3基础训练
- 写作Essay入门
- 口语Part 1-2基础训练

**家长配合**：
- 监督孩子每天30分钟词汇背诵
- 每周检查作业完成情况

### 第二阶段：专项突破（第7-12周）
**目标**：分模块强化，攻克各题型难点，提升应试技巧

**核心内容**：
- 综合教程Unit 7-12
- 阅读Part 4-7全题型突破
- 语用Part 3-4难点攻克
- 写作Part 2多文体训练（Article/Review等）
- 口语Part 3-4讨论与延伸
- 期中全真模考（第12周）

**家长配合**：
- 关注模考成绩分析
- 配合调整孩子学习重点

### 第三阶段：冲刺模考（第13-16周）
**目标**：全真模考，查漏补缺，调整到最佳考试状态

**核心内容**：
- 2次全真模考（官方真题）
- 模考详细讲评与错题分析
- 高频考点串讲
- 写作模板整理与背诵
- 口语高频话题演练
- 考前心态调整与注意事项

**家长配合**：
- 考前2周调整孩子作息
- 准备考试用品
- 给予心理支持

# 📋 方案B：高效强化方案

## 适合家庭
- 孩子基础扎实（PET优秀），自学能力强
- 能接受更高强度的学习节奏
- 希望在较短时间内集中突破
- 校内学业相对轻松，有时间投入

## 课程概览

| 项目 | 详情 |
|-----|------|
| **总时长** | 12周（约3个月） |
| **上课频率** | 每周1.5次（周二1.5小时 + 周六2小时） |
| **单次时长** | 1.5-2小时 |
| **总课时** | 42小时 |
| **上课形式** | 一对一 |

## 三阶段规划

### 第一阶段：快速入门与专项启动（第1-4周）
**目标**：快速熟悉FCE，直接进入专项训练，缩短适应期

**核心内容**：
- FCE考试结构精讲（1节课完成）
- 综合教程Unit 1-4精讲+快速练习
- 核心词汇加速积累（每周1.5个Unit）
- **提前启动**阅读Part 4-7和语用Part 3-4
- 写作Essay密集训练
- 口语Part 1-2强化

**特点**：
- 跳过部分基础重复内容
- 利用PET优秀基础，加快进度
- 每周1.5次课，保持学习连续性

### 第二阶段：高强度专项突破（第5-9周）
**目标**：高强度专项训练，全面提升应试能力

**核心内容**：
- 综合教程Unit 5-9
- 阅读全题型高强度训练
- 语用难点集中攻克
- 写作Part 2多文体精练
- 口语全流程模拟
- 期中模考（第9周）

**特点**：
- 每周作业量增加30%
- 增加在线答疑环节（每周1次，30分钟）
- 错题反复训练

### 第三阶段：密集模考与冲刺（第10-12周）
**目标**：通过密集模考快速找到问题并解决

**核心内容**：
- 3次全真模考（第10、11、12周各1次）
- 每次模考后详细讲评+针对性训练
- 高频考点快速串讲
- 应试技巧强化
- 考前状态调整

**特点**：
- 模考频率高，问题发现及时
- 最后阶段集中火力攻薄弱点

## 使用教材

与方案A相同，但使用方式不同：
- 综合教程精讲+快速练习，不重复讲解
- 专项教材使用量增加30%
- 真题使用量增加（3套→4-5套）

## 每周时间投入

| 项目 | 时间 |
|-----|------|
| 上课 | 3.5小时 |
| 词汇背诵 | 5小时（每天40-45分钟） |
| 听力作业 | 1小时 |
| 阅读/写作/口语作业 | 2小时 |
| 在线答疑（如需） | 0.5小时 |
| **总计** | **约12小时/周** |

## 预期目标

| 时间节点 | 目标水平 |
|---------|---------|
| 第4周末 | 完成FCE入门，开始专项训练 |
| 第9周末 | 各Part技巧熟练，期中模考Grade B以上 |
| 考前 | 稳定Grade B，冲击Grade A |

# 🎯 给家长的建议

## 如何选择？

**选择方案A，如果：**
- 孩子课业较忙，需要平衡多门学科
- 希望稳扎稳打，不追求快节奏
- 孩子性格偏谨慎，需要更多时间消化
- 预算相对有限

**选择方案B，如果：**
- 孩子PET优秀，基础扎实，学东西快
- 孩子自觉性好，能完成较高强度作业
- 希望更充分的准备，冲击更高分
- 校内学业相对轻松，有时间投入

## 我们的建议

基于该生**PET优秀**的成绩，说明：
1. 英语基础扎实，学习能力较强
2. 已经适应剑桥考试体系
3. 具备快速适应FCE难度的潜力

**初步推荐**：如果孩子能接受，**方案B**可以更好地发挥他的优势，在较短时间内达到更高水平。

但最终选择请结合：
- 孩子的时间安排和意愿
- 家庭的时间和经济预算
- 对成绩的期望目标

## 无论选择哪个方案，我们都会：

1. **定期反馈**：每2周提供学习进度报告
2. **模考分析**：每次模考后提供详细成绩分析
3. **随时沟通**：有问题随时通过微信/电话沟通
4. **灵活调整**：根据学习情况随时调整计划
