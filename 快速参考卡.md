# Coupang关键词监控脚本 - 快速参考卡

## 🚀 5分钟快速开始

### 1. 安装（1分钟）
```bash
bash /root/.openclaw/workspace/install_monitor.sh
```

### 2. 配置（1分钟）
```bash
vim ~/coupang_monitor/config.env
# 修改: REPORT_FILE="/path/to/你的报表.xlsx"
```

### 3. 运行（1分钟）
```bash
~/coupang_monitor/run.sh
```

### 4. 设置定时任务（2分钟）
```bash
~/coupang_monitor/setup_cron.sh
# 选择选项1: 每周一早上9点
```

## 🔧 常用命令速查

### 基本使用
```bash
# 分析单个报表
python3 coupang_keyword_monitor.py 报表.xlsx

# 指定输出目录
python3 coupang_keyword_monitor.py 报表.xlsx -o ./reports

# 静默模式（定时任务用）
python3 coupang_keyword_monitor.py 报表.xlsx -q

# 查看帮助
python3 coupang_keyword_monitor.py --help
```

### 批量处理
```bash
# 处理目录下所有报表
for file in /path/to/*.xlsx; do
    python3 coupang_keyword_monitor.py "$file" -o ./reports -q
done
```

### Cron定时任务
```bash
# 编辑定时任务
crontab -e

# 每周一9:00
0 9 * * 1 /usr/bin/python3 /path/to/coupang_keyword_monitor.py /path/to/报表.xlsx

# 每天9:00
0 9 * * * /usr/bin/python3 /path/to/coupang_keyword_monitor.py /path/to/报表.xlsx

# 每周一和周四9:00
0 9 * * 1,4 /usr/bin/python3 /path/to/coupang_keyword_monitor.py /path/to/报表.xlsx
```

## ❓ 常见问题速查

### Q: 报错 "ModuleNotFoundError"
```bash
pip3 install pandas openpyxl numpy
```

### Q: 报错 "FileNotFoundError"
- 检查文件路径是否正确
- 使用绝对路径：`/home/user/报表.xlsx`

### Q: 报表列名不匹配
- 确认报表包含：关键词、广告费、展示次数、点击次数、销量、销售额、商品名

### Q: 如何修改判断阈值？
编辑脚本中的 `classify` 函数，修改数值即可。

### Q: 如何添加飞书通知？
详见完整指南第7章，添加webhook调用代码。
