│  │ (定级)   │  │ (自适应) │  │ (待接入) │  │         │   │
│  └──────────┘  └──────────┘  └──────────┘  └────┬─────┘   │
└──────────────────────────────────────────────────┼──────────┘
                                                   │
                                                   ▼ API/手动录入
┌─────────────────────────────────────────────────────────────┐
│                     测评报告模板                            │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐   │
│  │ 基础信息 │  │ 能力雷达图│  │ 分项分析 │  │ 学习建议 │   │

### 1.2 数据映射表

| 测评系统输出 | 测评报告字段 | 转换逻辑 |
|--------------|--------------|----------|
| Part 1 正确率 | 听力理解得分 | 直接映射（Part 1主要测听力理解） |
| Part 2 难度路径 | 词汇/阅读得分 | H路径→高分，M路径→中分，跳过→低分 |
| 答题时长 | 学习力评估 | 超时→需提升专注力 |
| 错题类型分布 | 能力薄弱点 | 分析错误题型，定位问题 |
| 最终等级(A2/A1/入门) | 剑桥对标 | 直接填写 |

## 二、测评报告自动填充字段

### 2.1 可直接自动填充的区域

#### 表头信息
```
测评日期：系统自动生成（2025-03-25）
测评老师：KET自适应测评系统 / 人工复核
报告编号：XY-KET-20250325-001
```

#### 能力评分区
| 维度 | 分值范围 | 计算方式 |
|------|----------|----------|
| **听力理解** | 0-25分 | Part 1 得分 × 权重 |
| **词汇积累** | 0-20分 | Part 2 匹配正确率 × 权重 |
| **阅读理解** | 0-20分 | Part 2 理解程度评估 |
| **学习力** | 0-15分 | 答题时长 + 放弃率 |
| 口语表达 | 0-15分 | [待接入] 目前留白或人工评估 |
| 语音语调 | 0-5分 | [待接入] 目前留白 |

#### 剑桥对标区（自动勾选）
```
□ A2（KET水平）- 系统判定≥40分
□ A1（入门水平）- 系统判定25-39分  
□ Pre-A1（预备级）- 系统判定<25分
```

#### 错题分析（自动生成）
```
Part 1 错题分布：
- 直接信息题：错X道（如>2道，标记为薄弱环节）
- 意图推断题：错X道
- 逻辑推理题：错X道

主要薄弱点：________（自动填写错误率最高的题型）
```

### 2.2 需人工补充的区域

| 区域 | 原因 | 补充方式 |
|------|------|----------|
| 老师观察记录 | 系统无法观察非认知表现 | 试听课老师填写 |
| 口语表达评分 | 系统暂无口语模块 | 试听课老师评估 |
| 老师寄语 | 需要个性化情感连接 | 根据系统报告+面谈撰写 |
| 推荐课程 | 需结合家长意向 | 销售/老师根据报告推荐 |

## 三、测评系统改进方案

### 3.1 当前系统回顾

**已实现**：
- Part 1：18题抽6题，测试听力理解
- Part 2：3条分流路径（H/M/跳过）
- 50分制评分

**待接入**：
- Part 3-5（阅读/写作）
- 听力模块
- 口语模块

### 3.2 改进方向：从"分阶段"到"精细化自适应"

#### 方案对比

| 方案 | 逻辑 | 优点 | 缺点 | 推荐度 |
|------|------|------|------|--------|
| **A. 保持分阶段** | 按Part 1正确率粗分3档 | 简单，已实现 | 颗粒度粗，忽略具体错误类型 | ⭐⭐⭐ |
| **B. 题型细分自适应** ⭐ | 按错误题型动态选题 | 精准定位薄弱点 | 开发复杂度高 | ⭐⭐⭐⭐⭐ |
| **C. IRT项目反应理论** | 每题有难度参数，实时估算能力值 | 最科学精准 | 需要大量数据校准 | ⭐⭐⭐⭐ |

#### 推荐方案B：题型细分自适应

**核心逻辑**：不只是看"对几题"，而是看"错什么类型的题"

**Part 1 题型分类**（剑桥KET标准）：
| 题型代码 | 考察能力 | 难度 |
|----------|----------|------|
| T1 | 直接信息理解（时间/地点/人物） | ⭐ |
| T2 | 意图推断（Why/How） | ⭐⭐ |
| T3 | 逻辑推理（推断未明说的信息） | ⭐⭐⭐ |
| T4 | 词汇理解（生词/同义替换） | ⭐⭐ |
| T5 | 细节辨识（数字/颜色/动作） | ⭐ |

**自适应策略**：
```
IF T3/T4错误率 > 50%:
    → Part 2 选低难度（词汇+简单推理）
    → 推荐课程：基础夯实班

IF T1/T5错误多，T2/T3正确:
    → Part 2 选中难度（需关注细节+理解）
    → 推荐课程：能力提升班
    
IF 全部题型正确率 > 80%:
    → Part 2 选高难度（复杂推理+学术词汇）
    → 推荐课程：竞赛冲刺班
```

### 3.3 模块连接建议：独立但联动

**架构建议**：

```
┌─────────────────────────────────────────────────────────┐
│                    测评系统架构                          │
├─────────────────────────────────────────────────────────┤
│  模块层（独立开发，可插拔）                               │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐       │
│  │Part1-2  │ │Part3-5  │ │ 听力    │ │ 口语    │       │
│  │（已定级）│ │（阅读） │ │（已定级）│ │（待开发）│       │
│  └────┬────┘ └────┬────┘ └────┬────┘ └────┬────┘       │
│       └─────────────┴─────────────┴─────────────┘        │
│                         │                               │
│  自适应引擎层（统一调度）                                │
│  ┌─────────────────────────────────────────────────┐    │
│  │  · 根据已完成模块的错误类型调整后续模块难度       │    │
│  │  · 支持模块跳过（如时间不够或能力不匹配）        │    │
│  │  · 动态计算总分和能力画像                        │    │
│  └─────────────────────────────────────────────────┘    │
│                         │                               │
│  报告生成层                                             │
│  ┌─────────────────────────────────────────────────┐    │
│  │  · 汇总各模块数据                                │    │
│  │  · 生成测评报告PDF/打印版                        │    │
│  │  · 输出课程推荐建议                              │    │
│  └─────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────┘
```

**关键设计原则**：
1. **模块化**：Part 1-2、Part 3-5、听力、口语各自独立
2. **自适应连接**：通过"错误类型标签"连接各模块
3. **动态路径**：不是固定3条路，而是根据实时表现动态调整

### 3.4 具体改进 roadmap

#### 第一阶段（已完成）：基础自适应
- [x] Part 1 定级（3档分流）
- [x] Part 2 三套难度

#### 第二阶段（建议1个月内）：题型细分
- [ ] 给Part 1每道题打"题型标签"（T1-T5）
- [ ] 实现错误类型统计
- [ ] 根据错误类型推荐学习重点

#### 第三阶段（建议2个月内）：多模块联动
- [ ] 接入Part 3-5阅读题库
- [ ] 阅读模块也实现3档难度
- [ ] 建立"能力画像"（听力/词汇/阅读/语法四维）

#### 第四阶段（建议3个月内）：完整闭环
- [ ] 口语模块开发（AI评分或人工复核）
- [ ] 测评报告自动生成（PDF导出）
- [ ] 家长端小程序查看报告
