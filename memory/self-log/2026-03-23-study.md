# 自媒体学习笔记 - 2026年3月23日

## 一、2025-2026年三大平台算法核心变化

### 抖音：完播率降权，用户长期价值优先

**新推荐公式**：
```
推荐优先级 = 用户行为预测概率 × 行为价值权重
```

**权重变化对比**：
- 收藏（12分）→ 知识类×1.5，娱乐类×0.8
- 关注+追更（10分）→ 重视账号复访率
- 评论带观点（8分）→ 评论字数和互动深度
- 转发（6分）→ 转发好友数、二次打开率
- 完播（5分）→ 长视频×0.6，短视频×1.2
- 搜索（附加分）→ 知识类最多+3分

**核心变化**：
1. 收藏权重提升 → 知识密度高的内容更受推荐
2. 重视长期需求 → 收藏后复访、关注后追更行为被强化
3. 鼓励多元内容 → 打破"信息茧房"，跨领域内容获更多流量

**实操启示**：
- 放弃"短平快"思维，深度内容+用户认可度 > 单纯追求完播率
- 知识类账号可做45分钟长视频，只要收藏率和追更率高照样爆
- 发布"各地攻略"、"拍照技巧"等长尾价值内容，提升用户回访

---

### 小红书：标签匹配 + CES打分的阶梯式推荐

**CES评分公式**：
```
CES = 点赞（1分）+ 收藏（1分）+ 评论（4分）+ 转发（4分）+ 关注（8分）
```

**推荐流程**：
1. 种子推荐 → 根据内容+用户标签，推给粉丝和高匹配兴趣人群
2. CES打分 → 按互动质量评估
3. 阶梯式推荐 → 优质笔记进入更大流量池（含搜索流量）

**2025年平台新动作**：
- 上线「笔记声明原创」功能
- 打击同质化搬运内容
- 128个兴趣标签的AIGC识别
- 新增「内容稀缺性评分」
- 夜间21-23点本地内容曝光量提升37%

**实操建议**：
- 标签精准化：标题必带垂类关键词（如"干皮""不脱妆"）
- 互动设计：每条笔记结尾留钩子，引导评论（评论率目标5%）
- 原创强化：自己拍摄图片，标注"原创实拍"，文字800字+
- 搜索优化：标题加"2026""最新"等时间词

---

### 视频号：社交推荐权重加大

**流量来源公式**：
```
总曝光 = 私域基础曝光 × 社交裂变系数 + 公域推荐曝光
```

**2025年核心变化**：
- 社交推荐权重占比25%（抖音5%、小红书8%）
- "朋友点赞"权重提升，好友点赞内容更容易二次推荐
- 平台突出"朋友今天都在看"

**推荐链路**：
私域冷启动（社群/关注/朋友圈）→ 用户点赞 → 进入好友"朋友点赞页"和"推荐页" → 标签匹配+小流量测试 → 完播率/时长/互动率评估 → 进入更大流量池

**实操建议**：
1. 精准定向分享：结合好友标签，筛选分享人群（如美食内容分享给"美食达人"）
2. 内容植入社交动力：
   - 观点表达（"这就是我想说但不擅长说的话"）
   - 身份认同（"说出了XX群体的心声"）
   - 实用价值（"这个信息你得看看，对你有帮助"）
3. 可转发资产：视频结尾加总结页、清单、方法论三件套
4. 互动钩子：结尾设计引导评论

## 二、内容创作趋势与工具

### AI辅助创作已成标配
- 壹伴助手：AI全流程写作+排版（基于DeepSeek）
- 爱峰游：AI写爆款文案+爆款采集复刻+AI素材生成
- 多平台一键发布：支持20+主流平台

**关键提醒**：
- 平台已能精准识别AI拼凑内容并限制推荐
- 简单的复制粘贴AI内容会被降权
- 必须有"人的加工"和"原创性"

### 三大黄金赛道（TikTok/国内通用）
1. 萌宠类：治愈经济，全球通用语言
2. 生活技巧/干货类：实用价值，收藏率高
3. 垂类专业类：深度内容，追更率高

## 三、变现模式全览

- 垂直度是议价资本
- 深耕一个领域才有价值

### 3. 带货分佣

- 视频号：微信小店

## 四、核心洞察

> **"流量不是平台给的，而是用户给的。"**

三大平台算法趋同：
- 抖音：预测用户行为 + 长期价值
- 小红书：精准标签匹配 + 互动反馈
- 视频号：社交关系链 + 传播链路

**创作本质已从"迎合算法"转向"满足用户需求"**。

持续产出：高质量 + 高互动 + 高传播 = 赢得流量
