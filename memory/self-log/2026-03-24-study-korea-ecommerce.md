# 韩国电商学习笔记 - 2026-03-24

> 本次学习聚焦于韩国电商运营知识、平台规则、法律法规及税务条款，为Coupang平台业务开展提供基础认知。

---

## 一、韩国电商市场概况

### 1.1 主流平台
- **Coupang（酷澎）**：韩国最大电商平台，成立于2010年
  - 核心优势："火箭配送"（Rocket Delivery）——当日或次日达
  - 2022年Q4正式向中国卖家开放入驻
  - 不收取月租费，佣金按类目收取5%-10%
  - 支持中文客服服务

- **Naver Shopping**：依托韩国最大搜索引擎的电商平台

### 1.2 韩国消费者特点
- 网购前喜欢打电话咨询，重视客服体验
- 对配送时效性要求极高
- 偏好使用平台内置支付（可获得2%现金返还）

## 二、Coupang平台运营规则

### 2.1 入驻要求
**必备材料：**
- 公司营业执照（有效期≥1个月）
- 法人身份证/护照（有效期≥3个月）+ 手持证件照
- 银行账户证明（支持PingPong、Payoneer等第三方收款）
- 近6个月话费账单
- 其他平台交易流水（如有）

**禁售商品：**
- 酒精、烟草、药品
- 盗版/侵权产品
- 赃物、房地产、礼品券
- 需特殊认证但未获得认证的电子产品

### 2.2 物流政策（重要）
- **2022年12月30日起，Coupang不再允许自发货**
- 必须使用CGF（Coupang Global Fulfillment）或CGF Lite
- **CGF**：Coupang官方物流，入仓后由平台配送
- **CGF Lite**：入仓威海/深圳仓，平台负责跨境配送

**LCL拼箱服务（已记录在MEMORY.md）：**
- 威海线：70$/CBM，约6天
- 深圳线：98$/CBM，约8天

### 2.3 商品详情页规范
**禁止信息：**
- 配送时效、个人通关编码(PCCC)相关内容
- 退换货信息、售后服务内容
- "赠送"、"买一赠一"等活动描述（可用"3P"代替"1+1+1"）
- 外部平台链接（亚马逊、淘宝等）
- 友商平台logo
- 价格相关信息（直销价、工厂价等）
- "正品"字样
- 使用"斤"作为重量单位（韩国使用"千克"）

### 2.4 违规风险
- **售假**：直接封店，平台加强海外卖家审核
- **操纵评论**：禁止以奖励换取好评
- **跟卖风险**：Item Winner政策下，低价竞争者可能抢占流量
- **延迟发货**：可能导致产品下架或账户停售

## 三、韩国法律法规

### 3.1 核心法律框架
| 法律类别 | 主要法律 | 说明 |
|---------|---------|------|
| 电商经营 | 《电子商务法》 | 规范在线交易行为 |
| 外汇管理 | 《外汇交易法》 | 资金跨境流动规定 |
| 对外贸易 | 《对外贸易法》 | 2022年修订，建立电子文件交换系统 |
| 外商投资 | 《外国人投资促进法》 | 外资企业管理 |
| 消费者保护 | 《消费者保护法》 | 退换货、售后服务规定 |

### 3.2 企业设立流程
1. 外商投资申报（KOTRA或外汇银行）
2. 汇入投资资金
3. 法人设立登记（法院登记所）
4. 行业许可申请（如涉及化妆品、食品、医疗器械等）
5. **税务登记（营业开始后20日内，逾期罚款1%）**
6. 开立法人银行账户
7. 外商投资企业登记（出资完毕后60日内）

### 3.3 知识产权合规
- 严禁销售侵权产品（商标、版权、专利、设计权）
- 需确保产品认证齐全（电子产品需KC认证等）
- 未经授权不得销售品牌产品

## 四、韩国税务条款详解

### 4.1 税种体系
韩国税收分**国税**和**地方税**两大体系：

**国税：**
- 法人税（企业所得税）
- 附加价值税（增值税）
- 综合所得税（个人所得税）
- 关税

**地方税：**
- 购置税
- 财产税
- 地方所得税

### 4.2 增值税（VAT）- 核心
**税率：10%（标准税率）**

**纳税人类型：**
| 类型 | 年销售额标准 | 计税方式 |
|-----|------------|---------|
| 一般纳税人 | >8000万韩元 | 销项税额 - 进项税额 |
| 简易纳税人 | ≤8000万韩元 | 按行业增值率简易计税 |

**跨境电商特别规定：**
- **B2C模式**：外国卖家在韩国有第一笔销售即需注册增值税
- 年销售额>3亿韩元或设有实体仓库/办事处必须征收增值税
- 可选择直接征收（自行申报）或代征（委托韩国支付机构）
- **2025年起**：跨境电商年销售额超1亿韩元须注册增值税号，否则平台代扣10%

**计算公式：**
```
应纳税额 = 销项税额 - 进项税额
销项税额 = 销售额 × 10%
进项税额 = 采购成本中包含的增值税
```

**申报周期：** 每季度结束后25天内

### 4.3 法人税（企业所得税）
**2023年修订后税率：**

| 应纳税所得额 | 税率 | 累进抵扣额 |
|------------|-----|-----------|
| ≤2亿韩元 | 9% | 0 |
| 2亿-200亿韩元 | 19% | 2000万韩元 |
| 200亿-3000亿韩元 | 21% | 4.2亿韩元 |
| >3000亿韩元 | 24% | 9.42亿韩元 |

**申报要求：**
- 每季度按上年利润预缴
- 财年结束后3个月内完成全年申报

### 4.4 进口关税
**税率结构：**
- 基础税率：0%-40%（根据HS编码）
- 增值税：（CIF价 + 关税）× 10%
- 特殊调节税：反倾销税、反补贴税等

**免税额度：**
- 商品价值≤150美元（约20万韩元）：免征关税，但需缴增值税
- 超过150美元：全额征收关税+增值税

**中韩FTA优惠：**
- 满足区域价值成分≥40%可享优惠税率
- 部分工业品零关税

### 4.5 中韩税收协定
- **股息预提所得税**：通常税率5%-10%
- **利息预提所得税**：通常税率10%
- **特许权使用费**：通常税率10%
- 避免双重征税机制

### 4.6 税务合规要点
1. **及时登记**：营业开始后20天内完成税务登记
2. **账实一致**：货款与清关成本需保持一致（韩国税务监管严格）
3. **避免包税**：包税清关不使用韩国公司抬头，增值税无法抵扣
4. **季度申报**：增值税每季度申报一次
5. **保存凭证**：进出口单据、发票等需妥善保存

## 五、2024-2025年重要政策变化

### 5.1 外汇交易改革（2024年6月生效）
- 银行间外汇市场交易时间延长至次日2:00
- 外国投资者可透支韩元进行结算
- 放宽韩元账户限制，可通过ICSD进行结算
- 简化综合账户外汇交易流程

### 5.2 税收优惠政策（2024年税法修正案）
- 研发费用税收抵免期限延长至2027年12月31日
- 中小企业税收优惠期限由3年延长至5年（上市中小企业可延至7年）
- 股东回报提高的上市企业可享受法人税减免
- 废除原定于2025年实施的金融投资所得税

### 5.3 跨境电商监管加强
- 平台对海外卖家审核趋严
- 强制要求中国卖家使用平台物流代理服务
- 假货问题引发多平台紧急措施

## 六、实践建议

### 6.1 入驻前准备
1. 评估产品是否符合平台禁售规定
2. 确认是否有KC认证等必要认证
3. 准备完整的入驻材料（尤其注意证件有效期）
4. 选择合适的物流方案（CGF/CGF Lite）

### 6.2 运营中注意
1. **定价策略**：考虑Item Winner机制，但避免恶性低价竞争
2. **物流时效**：威海线6天、深圳线8天，预留充足库存
3. **税务合规**：按时申报增值税，保留完整财务凭证
4. **知识产权**：确保产品不侵权，避免封店风险

### 6.3 长期规划
1. 考虑在韩国注册法人以优化税务结构
2. 利用中韩FTA降低关税成本
3. 关注平台政策变化，及时调整运营策略
4. 建立本地化客服能力，提升用户体验
