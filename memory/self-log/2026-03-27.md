## 一、Coupang平台运营规则

### 1.1 入驻与店铺类型
- **本土店**：需在韩国有实体业务或分公司，需韩国公司主体
- **跨境店**：适合从其他国家向韩国销售的中国卖家
- **新入驻要求**（2024-2025）：中国大陆卖家需提交其他主流平台（Amazon/Shopee/eBay）最近6个月销售记录证明

### 1.2 物流模式
| 模式 | 说明 | 状态 |
|-----|------|------|
| CGF | Coupang全球物流服务（官方仓） | 推荐 |
| CGF Lite | 轻量化全球物流 | 推荐 |
| 自发货(FBM) | 卖家自行处理物流 | 2022年12月起已停止新注册 |

### 1.3 主图规范（重要！）
- **格式**：JPG、PNG（禁止GIF）
- **背景**：纯白色
- **占比**：商品占主图面积80%以上
- **位置**：居中、正面展示
- **禁止**：Logo、文字、拼接图、赠品展示
- **数量/重量**：如主图展示需与SKU实际设定一致

### 1.4 禁售产品清单
- 酒精类产品（韩国传统酒除外）
- 香烟、毒品
- 未经审查的视频/DVD/软件
- 侵犯知识产权商品
- 处方隐形眼镜/眼镜
- 未经许可的自卫产品（喷雾、电击枪）
- 房地产
- 游戏虚拟物品
- 对青少年有害的媒体产品
- 价值超1000万韩元的高端艺术品（无法验证来源）

### 1.5 KC认证要求
- **必须认证**：婴幼儿产品、电子电器、照明设备、电池等
- **申请限制**：部分KC认证仅可由韩国注册商业实体申请
- **平台要求**：WING系统需准确填写KC信息，验证通过后显示在商品详情页
- **违规处罚**：严重，可能下架或封号

## 二、韩国电商法律法规

### 2.1 核心法律框架

| 法律名称 | 实施时间 | 主要内容 |
|---------|---------|---------|
| 《电子商务基本法》 | 1999年 | 定义电子商务概念、电子文件要求、纠纷解决机制 |
| 《电子签名法》 | 1999年 | 数字签名法律效力、认证要求 |
| 《电子商务消费者保护法》 | 2002年（多次修订） | 消费者保护核心法律 |
| 《个人信息保护法》(PIPA) | 2020年修订 | 个人信息收集、使用、存储规范 |
| 《公平标签和广告法》 | 2014年 | 禁止虚假/夸大/欺骗性广告 |
| 《标准合同条例法》 | 2007年 | 防止不公平合同条款 |

### 2.2 消费者保护关键条款

**退货权利**：
- 7天无理由退货权（订单日期后7天内有效）
- 平台有义务配合政府召回令，回收销毁问题产品

**平台责任**：
- 对交易给消费者造成的损害承担连带责任
- 必须建立买卖家中介系统
- 有义务告知消费者展示产品是否为广告
- 跨境电商：需指定本地代理商处理纠纷

**信息披露要求**：
- 初始页面、产品描述页、订单确认页必须包含完整披露
- 必须提供条款条件和隐私政策链接
- 广告必须真实，禁止误导性表述

### 2.3 直播营销特别规定（2020年《网络直播服务法》）
- 直播平台需向韩国通信委员会注册
- 主播直播前需申报，提供销售商品信息
- 平台必须监管直播内容，发现虚假宣传应下架或停播
- 主播必须提供真实商品信息

### 2.4 违规处罚
- 3个月内违规10次 → 封号
- 6个月内违规20次 → 封号
- 罚款、禁止销售等

## 三、韩国税务条款

### 3.1 税种概览

| 税种 | 税率 | 说明 |
|-----|------|------|
| 增值税(VAT) | 10% | 主要流转税 |
| 进口关税 | 平均8%（因品类而异） | 农产品高、电子产品低 |
| 企业所得税 | 9%-24% | 根据利润阶梯计算 |

### 3.2 增值税(VAT)详解

**纳税人**：
- 年销售额 > 8亿韩元：一般纳税人（可抵扣进项）
- 年销售额 ≤ 8亿韩元：小规模纳税人（简易计税，可能无法抵扣）

**计算公式**：
```
应纳税额 = 销项税额 - 进项税额
销项税 = 销售额 × 10%
进项税 = 进货/服务支出 × 10%
```

**申报周期**：
- 季度申报（每季度结束后25天内）
- 预申报：每年3月、9月结束季度后25天内

**跨境电商特殊规定**：
- **B2C**：外国卖家在韩国有第一笔B2C销售即须注册VAT，负责代收代缴
- **B2B**：韩国买方企业按"反向征收"自行申报
- **电子服务**：外国公司向韩国提供电子服务需缴纳VAT

### 3.3 进口税务

**进口增值税**：
```
进口增值税 = (CIF价格 + 关税) × 10%
```
- 可用作后续销售环节抵扣
- ⚠️ 重要：必须使用自己公司税号清关才能抵扣
- ⚠️ "双清包税"无法抵扣，税务风险极高

**关税门槛**：
- 商品价值 < 150美元（约20万韩元）：通常免关税
- 但增值税仍需缴纳

**关税类型**：
- 基本税率
- 临时税率（政策调整）
- 弹性税率（反倾销、反补贴、季节性）
- 协定税率（FTA优惠，如韩美、韩欧）

### 3.4 企业所得税(法人税)

| 利润区间 | 税率 |
|---------|------|
| ≤ 2亿韩元 | 9% + 地方税0.9% |
| 2亿-200亿韩元 | 19% |
| 200亿-3000亿韩元 | 21% |
| > 3000亿韩元 | 24% |

**申报要求**：
- 每季度预缴（按上年利润）
- 财年结束后3个月内完成全年申报

### 3.5 中韩税收协定要点

**股息预提所得税**：
- 一般原则：按来源国法律征税
- 协定优惠：符合条件可降低税率

**避免双重征税**：
- 已在韩国缴纳的税款可在中国抵免
- 需提供完税证明

## 四、关键合规提醒

### 4.1 资金流一致性
韩国税务监管严格，**货款和进口清关成本必须保持一致**。

### 4.2 出口退税
- 出口商品适用零税率（0%）
- 需单独申报
- 需保留完整出口文件

### 4.3 发票要求
- 必须开具税务发票
- 需包含：注册号、名称、交易金额、税额、日期
- 外汇发票需按基本汇率转换为韩元

### 4.4 违规风险
- 使用他人税号清关 → 无法抵扣，涉嫌违规
- 申报数据与平台报送数据差异过大 → 税务预警
- 未按时申报 → 罚款
