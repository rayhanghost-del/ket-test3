## 一、Coupang平台运营知识

### 1.1 平台优势与核心服务

**火箭配送 (Rocket Delivery)**
- 24小时内送达服务
- 使用官方物流可获得"火箭配送"标签
- 订单转化率提升28%

**自发货 (FBM) vs 官方物流**
- FBM: 卖家自行负责全部物流环节，平台设定6天配送缓冲期（不含周日和韩国法定节假日）
- 官方物流: 首尔圈当日达、地方城市次日达，基准运费首重1kg以内8500韩元

**LCL拼箱服务 (Coupang Direct LCL)**
- 威海线: 70$/CBM，约6天
- 深圳线: 98$/CBM，约8天
- 报价包含: 集货仓操作费、海运费、清关费、FC配送费
- 报价不含: 国内运输、关税/税费、增值服务

### 1.2 入驻与审核

**两种开店模式**
- 极速开店: 流程简化，最快1周下店
- 普通开店: 审核更严格，周期较长

**五维审核体系**
1. 初审: 企业资质与商品合规性
2. 复审: 价格竞争力（要求低于主流平台均价7%-9%）、售后服务
3. 技术审核: API接口与数据安全
4. 品牌审核: 知识产权状况
5. 终审: 神秘客检测物流响应时效
- 审核周期: 7-10个工作日

**保证金要求**
- 企业卖家: 1500万韩元（2024年下调）
- 个人卖家: 2000万韩元

### 1.3 平台费用结构

**佣金率（阶梯佣金+技术服务费）**
- 3C数码类: 14.5%（含5%技术服务费）
- 家居日用类: 11.8%（含3%技术服务费）
- GMV超10亿韩元/月: 佣金率下调1.2-2.2个百分点
- 跨境卖家额外收取1.8%跨境服务费

**广告投放**
- 首页推荐位CPC出价中位数: 3200韩元
- 搜索结果页顶部CPM报价: 7.5万韩元
- 热门品类CPC可达4800韩元

**退货处理费**
- 非质量问题退货: 运费+订单金额4.5%操作费

### 1.4 流量与转化

**自然流量算法**
- 双因子模型: 价格竞争力（65%权重）+ 用户行为（35%权重）
- 4.9星以上商品点击率是3.5星商品的3.8倍

**排名优化**
- 标题需包含3-5个核心关键词
- 30天价格稳定期: 频繁调价导致搜索权重下降22%

### 1.5 运营要点

**商品上架要求**
- 单箱重 ≤30kg，长宽高之和 ≤2500mm，单边 ≤1100mm
- 必须贴: 产地标（MADE IN CHINA）、Coupang条形码、韩文标签、入仓外箱单

**评价与测评**
- 核心是积累真实口碑，非刷单
- 严禁虚标原价: 2024年Q2下架1.1万条违规商品链接

**退货规则**
- 14日无理由+90日质量保障
- 低于5万韩元商品可建议丢弃；高于5万韩元需在3天内安排取回

**结算周期**
- 月度结算制度，次月第15个工作日支付
- 最低支付门槛: 当月GMV超过100万韩元

## 二、韩国税务合规

### 2.1 增值税 (VAT) - 核心税种

**标准税率: 10%**
- 适用于绝大多数商品和服务
- 出口商品和劳务适用零税率

**纳税人分类**
- 一般纳税人: 年销售额超过8亿韩元，必须按正常流程申报
- 小规模纳税人: 年销售额≤8亿韩元，可选择简易计税（但可能无法抵扣进项税）

**计算公式**
```
应纳税额 = 销项税额 - 进项税额
销项税 = 销售额 × 10%
进项税 = 采购成本 × 10%
```

**申报周期**
- 季度申报: 每季度结束后25天内
- 小微企业可季度申报

**跨境电商特别规定**
- B2C: 外国卖家在韩国有第一笔B2C销售就必须注册增值税
- B2B: 韩国买方企业按"反向征收"申报缴税

### 2.2 法人税（企业所得税）

**累进税率结构**（2023年下调后）
| 应纳税所得额 | 税率（含地方税） |
|-------------|-----------------|
| ≤2亿韩元 | 9% + 0.9% = 9.9% |
| 2亿-200亿韩元 | 19% + 1.9% = 20.9% |
| 200亿-3000亿韩元 | 21% + 2.1% = 23.1% |
| >3000亿韩元 | 24% + 2.4% = 26.4% |

**申报要求**
- 季度预缴: 按上一年利润预缴
- 年度汇算: 财年结束后3个月内完成

### 2.3 进口关税

**税率水平**
- 平均关税约8%，不同品类差异大
- 高关税: 农产品（大米、牛肉）、部分纺织品、汽车
- 低/零关税: 电子产品、机械设备、自贸协定商品

**计算基数**
```
关税 = CIF价格 × 关税税率
CIF = 商品价格 + 国际运费 + 保险费
```

**包裹免税门槛**
- 价值低于150美元（约20万韩元）: 免关税，但增值税仍需缴纳

**特殊规定**
- 锂电池（188特殊规定）: 需提供MSDS认证、非危险品运输鉴定报告、UN38.3测试报告

### 2.4 进口增值税与销售增值税的关系

**关键点**: 进口增值税可抵扣销售增值税

```
进口环节: 缴纳关税 + 进口增值税（10%）
销售环节: 收取销售增值税（10%）
实际缴纳: 销售增值税 - 进口增值税（可抵扣部分）
```

**重要提醒**
- 不走包税: 包税清关不使用韩国公司抬头，进口增值税无法抵扣
- 货款和进口清关成本必须保持一致（韩国税务监管严格）

### 2.5 DDP条款下的税务处理

**DDP (完税后交货)**
- 卖方承担目的国进口税（关税+增值税）
- 费用包含在报价中，与货款一起收取

**增值税退税问题**
- 卖方支付增值税无法退税
- 卖方在目的国注册公司并以该公司名义进口缴税，可申请退税

## 三、韩国电商法律法规

### 3.1 公司注册形式

**株式会社 (주식회사)**
- 相当于股份有限公司
- 最少需2人注册，需1名监事
- 每三年法人代表和监事需重新办理续任公证认证
- 可发行证券股票，可转让和抵押

**税务登记要求**
- 在税务局注册，获得税务识别号
- 注册增值税和企业所得税

### 3.2 平台合规要求

**税务合规**
- 强制要求卖家申报韩国增值税
- 未注册企业将被限制参与平台促销活动
- 需提供韩国当地公司注册证明或通过指定代理完成合规认证

**新《电商法》规定**（2024年修订）
- 跨境电商必须提供本土客服
- 订阅服务需单独确认，否则面临高额罚款

**KC认证**
- 产品上架时必须准确选择并填写KC认证信息
- 通过系统验证后方可销售

### 3.3 知识产权保护

**商标法修订**（2025年）
- 故意商标侵权惩罚性赔偿上限从3倍提高至5倍
- 侵权风险显著增加

**专利与版权**
- 需确保产品不侵犯韩国本土专利权
- 图片、描述需原创或获得授权

### 3.4 消费者权益保护

**退货规则**
- 14日无理由退货
- 90日质量保障
- 非质量问题: 买家承担双向运费

**虚假宣传处罚**
- "零容忍"政策
- 虚标原价将被下架并扣减保证金

### 3.5 数据与隐私

**电子发票系统**
- 所有企业交易必须通过电子发票系统开具和接收
- 实现税务透明化
- 所有交易需官方系统留痕

### 3.6 海关与通关

**2024年政策更新**
- 2024年7月: 简易出口申报对象限额由200万韩元提升至400万韩元
- 2024年8月起正式实施

**RCEP红利**
- 中韩已实施第三轮降税
- 最终零关税产品比例达86%
- 50.4%产品在协定生效当日享零关税

## 四、实用合规建议

### 4.1 入驻策略

**跨境店 vs 本土店**
- 跨境店: 初期可自发货，无需缴纳韩国增值税（但需遵守中国税务规定）
- 本土店: 可使用火箭仓配送，需严格缴纳10%增值税

**金标认证**
- 提供韩国海关AEO认证或签订平台指定物流协议
- 订单履约率达97.2%
- 享受专属流量扶持和0.8%佣金优惠

### 4.2 税务优化

**税收协定利用**
- 中韩《关于对所得避免双重征税和防止偷漏税的协定》
- 股息、利息、特许权使用费可享受优惠税率

**税收优惠政策**
- 符合条件的外资企业: 前3年免缴法人税、后2年减半
- 首尔以外地区投资: 最高50%地方税减免

### 4.3 风险规避

**常设机构（PE）风险**
- 在韩国设立办公室、仓库或长期雇佣员工可能构成PE
- 需就韩国全部利润缴纳企业所得税

**关联交易定价**
- 确保符合独立交易原则
- 准备好转让定价文档

**税务欺诈风险**
- 避免虚报成本、隐瞒收入、使用不合规发票
- 韩国税务监管严格，违规后果严重

### 4.4 操作建议

1. **聘请专业顾问**: 财务顾问或税务顾问协助理解复杂税务法规
2. **准确记录**: 保存所有交易数据、凭证、发票
3. **及时申报**: 按时完成季度/年度税务申报
4. **关注政策变化**: 韩国跨境电商税收政策持续更新
5. **合规运营**: 所有交易通过官方系统留痕，确保账目真实
