## 【问题案例1】택배사 배송지연으로 판매자 점수 0점 처리됨

**场景描述**：
판매자는 정시에 발송했으나, 택배사 사정으로 배송이 예상일보다 하루 늦어졌고, 이로 인해 판매자 점수(배송 부문)가 갑자기 0점이 되었다. 판매자는 정시배송을 완료했음에도 불구하고 택배사의 지연으로 패널티를 입게 되었다.

**问题分析**：
- 쿠팡의 판매자 점수 시스템은 "배송 완료일" 기준으로 판단되며, 판매자의 발송 시점은 반영되지 않음
- 택배사의 지연은 판매자 귀책 사유로 처리됨
- 쿠팡은 판매자가 직접 온라인 문의를 통해 소명 요청을 해야만 점수를 복구해줌
- 이는 시스템적으로 판매자와 쿠팡 모두의 시간을 낭비하는 비효율적인 프로세스임

**解决方案**：
1. 쿠팡 윙에 로그인 후 아래 링크로 접속
   - https://helpseller.coupang.com/hc/ko/requests/new?ticket_form_id=360000545712
2. "판매자점수 제외 - 정시배송완료" 선택
3. 필요 정보 입력:
   - 주문번호
   - 박스ID(=묶음배송번호)
   - 발송 시간 캡처 화면 첨부 (선택사항)
4. 제출 후 답변 대기 (보통 며칠 내 처리)

**关键要点**：
- ✅ 발송 시점을 꼭 캡처해서 보관할 것 (증빙 자료)
- ✅ 택배사 지연은 판매자 귀책이 아니므로 반드시 소명 요청할 것
- ✅ 정시배송 완료 증거가 있으면 점수 복구 가능

**原文链接**：https://nomadel.tistory.com/entry/쿠팡-정시발송했는데-판매자-점수가-갑자기-0이-되었다-판매자-점수-복구-후기

## 【问题案例2】단순 변심 반품을 판매자 귀책으로 접수당함

**场景描述**：
구매자가 단순 변심으로 반품을 신청했으나, 귀책사유를 '판매자'로 선택했고, 판매자는 이로 인해 왕복 배송비를 부담하게 되었다. 쿠팡 시스템에서는 구매자가 선택한 귀책사유대로 처리되어 판매자가 피해를 보는 경우가 빈번함.

**问题分析**：
- 구매자가 귀책사유를 '고객'으로 선택 → 반품 배송비는 구매자 부담 (결제금액에서 차감)
- 구매자가 귀책사유를 '판매자'로 선택 → 반품 배송비는 판매자 부담 (쿠팡이 지급하지 않음)
- 일부 구매자는 의도적으로 '판매자'를 선택하여 반품비를 회피
- 판매자가 방어하지 않으면 그대로 손해를 봄

**解决方案**：
1. **초기 대응 (가장 중요)**:
   - 구매자에게 연락하여 상황 설명
   - 귀책사유를 '고객'으로 변경 요청 혹은 직접 반품비 송금 요청
   - 채팅 내역이나 통화 기록을 증빙으로 남겨둘 것

2. **협의 실패 시 쿠팡 확인 요청**:
   - 입고 후: WING > 반품 관리 > 쿠팡 확인 요청
   - 입고 전: WING > 온라인 문의 > 확인 요청 접수하기
   - 최소 2영업일 이상 구매자와 연락 시도했다는 증빙 필요
   - 상품 회수 후 168시간(7일) 안에 접수해야 함

**关键要点**：
- ✅ 반품 접수 즉시 확인하고 당일 대응할 것
- ✅ 구매자와의 모든 소통 내역은 스크린샷으로 보관
- ✅ 쿠팡 확인 요청은 회수 후 7일 이내에 해야 함
- ✅ 고객 귀책으로 인한 반품은 반드시 비용을 받아내야 손해 방지 가능

## 【问题案例3】AI 광고로 인해 상관없는 키워드에 광고비 낭비

**场景描述**：
판매자가 'AI스마트광고'나 '매출최적화광고'를 사용했는데, 쿠팡 AI가 '구두' 상품에 '운동화' 키워드를 매칭하거나, '셔츠' 상품에 '맨투맨' 키워드를 매칭하는 등 상품과 무관한 키워드에 광고비가 낭비됨. 한 판매자는 최대 1,900만 원까지 손해를 봤다고 함.

**问题分析**：
- AI스마트광고와 매출최적화광고는 키워드를 직접 추가할 수 없음 (제외만 가능)
- 쿠팡 AI가 상품명+카테고리+검색태그+옵션을 바탕으로 키워드를 자동 매칭
- AI의 예측이 부정확하여 비효율적인 키워드에 광고비가 소진됨
- 상품별 ON/OFF 불가능하여 주력 상품 집중 투자 어려움

**解决方案**：
1. **제외 키워드 설정**:
   - 광고 캠페인 분석 → 비효율 키워드 식별
   - 상품과 무관한 키워드는 제외 키워드로 등록
   - 정기적으로 제외 키워드 리스트 업데이트

2. **광고 유형 선택**:
   - 상품이 많은 위탁판매 → 수동성과형광고 추천 (직접 키워드 설정)
   - 시간이 없는 경우 → AI스마트광고 (제외 키워드 설정 필수)
   - 공격적 마케팅 원할 경우 → 매출최적화광고 (키워드 제외 기능 활용)

3. **효율화 도구 활용**:
   - 세이비 등 광고 효율화 서비스 활용
   - 돈 먹는 키워드 자동 식별 및 제거 기능 사용

**关键要点**：
- ✅ 자동 광고는 반드시 제외 키워드 설정이 필수
- ✅ 2주에 1번 정도 키워드 성과 분석 필요
- ✅ 상품과 무관한 키워드는 즉시 제외
- ✅ ROAS(광고 수익률) 350% 정도를 기준으로 설정 후 조정

## 【问题案例4】노출상품 ID 불일치로 상품 분리 노출

**场景描述**：
같은 상품의 5개 옵션 중 3개는 A 페이지에, 2개는 B 페이지에 노출되어 구매자가 한 페이지에서 모든 옵션을 선택할 수 없게 됨. 더 심각한 경우 다른 판매자의 상품에 내 옵션이 묶이거나, 내 상품 중 전혀 다른 상품에 옵션이 묶이는 문제가 발생.

**问题分析**：
- 쿠팡 시스템 오류로 인해 같은 상품의 옵션들이 다른 '노출상품 ID'로 분리됨
- 등록상품 ID ≠ 노출상품 ID (노출상품 ID는 실제 판매 페이지 URL에 사용됨)
- 옵션이 분리되면 구매자가 다른 옵션을 선택할 수 없어 구매 전환율 하락
- 다른 판매자 상품에 옵션이 묶이면 매출이 갑자기 끊김

**解决方案**：
1. **문제 확인**:
   - 쿠팡 WING > 상품관리 > 상품조회/수정 > 상품목록
   - 상품 좌측 화살표 클릭 → 등록상품 ID와 노출상품 ID 확인
   - 같은 상품의 옵션들이 서로 다른 노출상품 ID를 가지고 있는지 확인

2. **해결 방법**:
   - 쿠팡 윙 → 온라인문의 → 상품관리 → 상품수정요청 → 동일한 상품이니 하나의 상품 ID로 결합해주세요
   - 또는 직접 문의 페이지: https://help.seller.coupang.com
   - 대량 결합이 필요한 경우 엑셀 파일 다운로드하여 작성 후 제출

3. **처리 기간**:
   - 최대 8일까지 소요될 수 있음

**关键要点**：
- ✅ 상품 등록 후 반드시 노출상품 ID 일치 여부 확인
- ✅ 옵션이 여러 페이지로 분리되면 즉시 결합 요청
- ✅ 다른 판매자 상품에 옵션이 묶이면 아이템위너 신고를 통해 분리 가능
- ✅ 상표권 등록 후 제품/포장에 브랜드 표시 → 향후 매칭 방지 가능

**原文链接**：https://infjblanc.com/쿠팡-노출상품-id/

## 【问题案例5】판매자격 정지 및 회복 과정

**场景描述**：
정상적으로 판매하던 판매자가 어느 날 '고객 만족을 해치는 행위'라는 모호한 사유로 판매자격 정지를 당함. 이메일에는 어떤 특정 건이나 고객과의 상담 과정에서의 문제로 인해 정지되었다는 구체적인 내용이 없어 원인 파악이 어려움.

**问题分析**：
- 쿠팡은 '고객 만족을 해치는 행위'라는 추상적인 사유로 판매자격을 정지함
- 어떤 특정 건으로 인해 정지되었는지 구체적으로 알려주지 않음
- 판매자는 자신이 어떤 잘못을 했는지 파악하기 위해 과거 클레임 기록을 모두 검토해야 함
- 60% 정도의 상품만 판매정지되는 경우도 있어 더욱 혼란스러움

**解决方案**：
1. **즉시 조치**:
   - 확약서 및 개선계획서 작성 (이메일에 첨부된 파일)
   - 과거 고객 클레임 관련 정보 취합
   - 쿠팡 담당자에게 연락하여 원인 파악 시도

2. **문서 제출**:
   - 확약서: 재발 방지를 약속하는 서류
   - 개선계획서: 구체적인 개선 방안 작성
   - 해당 서류를 이메일로 제출 후 내부 검토 대기

3. **원인 파악 방법**:
   - 직원들과 가장 최근 발생한 고객 클레임 관련 정보 회의
   - 고객이 잘못 주문 후 무상 교환 요구하며 반품 거부했던 사건 등 확인
   - 쿠팡 담당자와 통화하여 어감상 원인 추론

**关键要点**：
- ✅ 중요한 이메일은 매일 확인할 것 (퇴근 무렵 발송된 메일도 놓치지 말 것)
- ✅ 고객 클레임 발생 시 상세한 기록 보관 (대응 증거 자료)
- ✅ 확약서와 개선계획서는 성의껏 작성하여 신속히 제출
- ✅ 쿠팡 담당자와의 연결고리를 미리 만들어두는 것이 중요

## 【问题案例6】쿠팡그로스 광고비 지출 대비 ROAS 저조

**场景描述**：
판매자가 쿠팡그로스(광고 플랫폼)를 통해 신규 상품 노출을 시도했으나, 클릭당 비용(CPC)은 계속 나가는데 실제 매출로 이어지지 않아 적자를 봄. 인기 키워드는 클릭당 200~500원 이상으로 올라가 광고 예산이 빠르게 소진됨.

**问题分析**：
- 쿠팡은 광고가 없으면 노출이 거의 되지 않는 구조
- 신규 셀러나 신제품은 자연 노출로 경쟁이 어려워 광고 의존할 수밖에 없음
- 상위 노출되어도 실제 구매는 가격이 더 싼 상품으로 이동하는 경우가 많음
- 마진이 낮은 상품은 광고를 돌릴수록 손해
- "광고 안 돌리면 매출 0원. 돌리면 팔리는데 남는 게 없음"

**解决方案**：
1. **ROAS 기반 키워드 관리**:
   - 광고 수익률(ROAS) 기준으로 키워드별 성과 체크
   - 비효율 키워드는 제외 또는 입찰가 하락
   - 전환율 높은 키워드에 집중 투자

2. **마진 계산**:
   - 광고비를 포함한 원가 구조 정확히 분석
   - 마진이 충분한 상품만 광고 진행
   - 목표 ROAS 현실적으로 설정 (처음엔 350% 권장)

3. **소량 테스트 후 확장**:
   - 처음부터 많은 예산을 쓰기보다 A/B 테스트 진행
   - 효율적인 키워드 확인 후 예산 확대

4. **상품력 보완**:
   - 광고만으로는 장기 생존 어려움
   - 리뷰 50개 이상, 평균 평점 4.5 이상 목표
   - 상품력, 브랜딩 강화

**关键要点**：
- ✅ ROAS 분석은 필수 (안 하면 망함)
- ✅ 마진이 없는 상품은 광고하지 말 것
- ✅ 상위 노출 ≠ 구매 전환 (가격 경쟁력 확인 필요)
- ✅ 광고는 보조 수단, 상품력이 핵심

**原文链接**：https://loveeatpray.tistory.com/entry/쿠팡그로스의-현실-셀러들이-마주한-진짜-이야기

## 【运营技巧】쿠팡 빠른 정산(셀러월렛) 활용법

**适用场景**：
- 정산 주기가 느려 자금 압박을 겪는 판매자
- 추가 재고 확보를 위해 신속한 자금 회전이 필요한 경우
- 은행 대출이나 신용카드 할부로 이자 부담이 큰 판매자

**具体操作**：
1. **신청 방법**:
   - 쿠팡 윙 → 정산 관리 → 빠른 정산(셀러월렛) 신청
   - 쿠팡 셀러 전용 체크카드 발급 (하나은행과 협업)

2. **정산 방식**:
   - 전날 구매확정된 매출 금액의 90%를 다음날 오전에 미리 정산
   - 체크카드에 충전되는 형태로 제공
   - 추가 수수료나 이자 비용 없음 (무료)
   - 나머지 10%는 기존 정산 주기에 따라 나중에 입금

3. **사용 방법**:
   - 쿠팡 셀러 체크카드로 바로 사용 가능
   - 사용하지 않은 잔액은 정산일에 셀러 통장으로 입금

**效果预期**：
- 기존 19~60일 정산 주기에서 다음날 90% 정산으로 자금 회전 개선

2. **정산이 가장 큰 리스크**: 쿠팡의 느린 정산 주기(최대 60일)는 판매대금이 제때 돌지 않아 사업 자금이 막히는 '돈맥경화'를 유발한다. 충분한 운영 자금이 없으면 매출이 늘어날수록 오히려 망할 수 있다.
