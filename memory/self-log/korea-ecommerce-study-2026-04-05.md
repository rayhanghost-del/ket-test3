# 韩国电商学习笔记

> 学习时间：2026年4月5日 02:00-04:00  
> 学习主题：韩国电商运营知识、法律法规、税务条款

---

## 一、韩国电商市场概况

### 1.1 主流平台
- **Coupang（酷澎）**：韩国最大电商平台，被称为"韩国亚马逊"
  - 拥有庞大的消费人群和丰富商品类别
  - 超过50%的韩国网购者使用Coupang
  - 2022年起不再接受自发货，必须使用CGF/CGF Lite物流

### 1.2 入驻要求
**必备资料：**
- 企业营业执照
- 税务登记证明
- 商品资质证书
- 品牌授权书（销售品牌商品时）
- 稳定的供应链资源证明

**入驻流程：**
1. 注册商家账号
2. 提交入驻申请资料
3. 平台审批（通过后发入驻邀请）
4. 签订入驻协议并缴纳费用
5. 店铺装修
6. 上架产品

### 1.3 费用结构
- **入驻费用**：根据店面规模与行业而定
- **销售提成**：按比例收取的平台服务费
- **推广费用**：可选，提升商品曝光度
- **仓储费用**：使用Coupang仓储服务时产生

## 二、运营要点

### 2.1 产品上架策略
- **新店建议**：前期上架6-7个产品为宜，避免无销量产品拉低流量
- **无销量影响**：30天内无销量的产品会对店铺流量产生负面影响

### 2.2 标题优化
- 使用后台推广工具 → 搜索行业分析 → 搜索词分析
- 输入产品核心词获取100个相关词汇
- 优先选择热度高的关键词提升曝光

### 2.3 测图测款
- 每个款式累积测试点击至少500次以上
- 持续测试3-4天
- 关注图片点击率、款式转化率
- 数据不明显则延长测试时间，表现不佳及时更换

### 2.4 评价积累
- 0销量0评价产品难以吸引真实销量
- 前期建议积累100-200个评价为销售奠定基础

### 2.5 重要规则变更
- **SKU曝光规则**（2022年8月起）：多选项产品只曝光相同或更低单价的选项
- **主图要求**：有严格的质量和格式要求
- **CGF/CGF Lite**：自发货时代结束，必须使用平台指定物流

## 三、法律法规框架

### 3.1 核心法律
**《电子商务等消费者保护法》（2024年修订）**
- 2024年8月19日国务会议通过修订案
- 海外电商必须指定韩国境内代理人
- 代理人负责处理消费者投诉和争端
- 需在网站上公开代理人信息（姓名、地址、电话、邮箱）

**《个人信息保护法》(PIPA)**
- 规范个人信息的收集、使用、传播、存储和销毁
- 需获得用户同意才能收集和使用个人信息
- 个人有权要求查看、更正或删除自身信息
- 信息泄露需及时通知受影响者和当局
- 违规可能面临罚款、赔偿甚至刑事责任

**《公平标签和广告法》**
- 禁止虚假、夸大、欺骗性广告
- 禁止不公平比较和诽谤性广告
- 违反者将受到公平交易委员会查处

### 3.2 信息披露要求
根据《电子商务法》和《个人信息保护法》：
- 网站必须提供条款和条件链接
- 必须提供隐私政策链接
- 初始登录页、产品描述页、订单确认页必须包含所需披露项目

### 3.3 消费者保护义务
- 建立并维持接受消费者投诉的机构
- 对因电商遭受的损害提供赔偿
- 采取必要措施维护消费者信息安全
- 电子支付时需通知消费者

## 四、税务条款详解

### 4.1 增值税 (VAT)
**税率：** 10%（标准税率）

**征收条件：**
- 年销售额超过3亿韩元（约260,000美元）
- 在韩国境内有实体仓库或办事处

**计算方式：**
```
进口增值税 = (CIF价格 + 关税) × 10%
```

**申报周期：**
| 计税期间 | 申报及缴纳期间 | 申报对象 |
|---------|--------------|---------|
| 1-3月 | 4月1日-25日 | 法人 |
| 4-6月 | 7月1日-25日 | 法人和个体工商户 |
| 7-9月 | 10月1日-25日 | 法人 |
| 10-12月 | 次年1月1日-25日 | 法人和个体工商户 |

**征收方式：**
- **直接征收**：向韩国税务局申请登记，自行申报缴纳
- **代征**：委托韩国境内支付机构代为征收缴纳

### 4.2 关税
**税率范围：** 0% - 20%（根据商品类型、HS编码、原产地不同）

**计算公式：**
```
关税 = 关税征税标准 × 关税税率
```

**优惠措施：**
- FTA（自由贸易协定）成员国进口商品可能享有关税减免或豁免

**注意事项：**
- 关税为进口成本，不可用于后续销售抵扣
- 必须使用自己的公司税号清关才能抵扣进口增值税
- 使用"双清包税"渠道无法抵扣，税务风险极高

### 4.3 企业所得税（法人税）
**税率分级：**
| 利润范围 | 税率（含地方税） |
|---------|----------------|
| 不超过2亿韩元 | 9% + 0.9% = 9.9% |
| 2亿 - 200亿韩元 | 1800万韩元 + 超过部分19% |
| 200亿 - 3000亿韩元 | 37.8亿韩元 + 超过部分21% |
| 超过3000亿韩元 | 625.8亿韩元 + 超过部分24% |

**计算方式：**
```
营业利润 = 收入 - 货品成本 - 物流成本 - 平台费用 - 广告费用 - 其他杂费
```

### 4.4 消费税
**适用商品：** 酒类、烟草、奢侈品等特定商品

**计算依据：** 根据商品数量、重量或价值计算

### 4.5 零税率情形
以下情况适用增值税零税率（进项税额可退还）：
- 货物出口
- 发生在韩国境外的劳务
- 国际运输服务（船舶、飞机）
- 为取得外汇收入的其他货物劳务行为

## 五、合规建议

### 5.1 税务合规
1. **使用自己公司税号清关**，确保进口增值税可抵扣
2. **避免双清包税**，防止无法抵扣和税务风险
3. **货款和进口清关成本保持一致**，韩国税务监管严格
4. **按季度申报增值税**，小微企业可按季度申报
5. **准确记录销售、采购、库存等业务数据**

### 5.2 法律合规
1. **指定韩国境内代理人**（如达到销售规模标准）
2. **完善隐私政策**和用户信息保护措施
3. **确保广告真实**，避免虚假夸大宣传
4. **建立消费者投诉处理机制**

### 5.3 运营建议
1. **选择合适的产品类别**：3C电子、小家电、收纳户外等有机会
2. **重视物流**：利用Coupang火箭仓提升配送效率
3. **合规积累评价**：通过优质服务和产品获得真实好评
4. **持续测款上款**：不断测试找出畅销款

## 七、学习总结

韩国电商市场以Coupang为主导，具有巨大潜力但监管严格。成功进入韩国市场需要：

1. **合规先行**：了解并遵守《电子商务法》《个人信息保护法》等法规
2. **税务规划**：合理规划增值税、关税、企业所得税，使用正规清关渠道
3. **本地代理**：达到一定规模需指定韩国境内代理人处理消费者事务
4. **运营策略**：测款选品、优化标题、积累评价、合规推广

**关键记忆点：**
- 增值税率10%，按季度申报
- 关税0%-20%，必须用自身税号清关才能抵扣
- 企业所得税分级征收，最高24%
- 2024年新法要求海外电商指定韩国境内代理人
