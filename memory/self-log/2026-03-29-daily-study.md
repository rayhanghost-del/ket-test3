## 【问题案例1】묶음상품 단위가격이 아이템위너 선정에 미치는 영향

**场景描述**：一位쿠팡셀러销售500ml包装水，10个一组销售。虽然总价比竞争对手低，但始终无法成为아이템위너(商品获胜者)，导致流量和销量大幅下降。

**问题分析**：
- 2024年7月28日起，쿠팡新增"상품 단위가격(单位价格)"作为아이템위너 선정 기준
- 该卖家销售的10个装单位价格为每100ml 200원，而竞争对手的单个装单位价格为每100ml 180원
- 尽管10个装的总价更低，但단위가격이 높아 아이템위너 선정에서 불리

**解决方案**：
1. 进入쿠팡윙 → 가격관리 페이지
2. 点击'쿠팡추천가격 적용'按钮自动调整
3. 或在'내 상품가'中手动修改단위가격
4. 修改后쿠팡会进行实时审核，审核通过后重新恢复曝光

**关键要点**：
- 단위가격 = 商品总价 ÷ 总容量或数量（如ml、kg等）
- 묶음상품(捆绑商品)的단위가격高于小包装时，아이템위너竞争力下降
- 단위가격过高的商品会在"구매옵션(购买选项)"中被限制曝光

## 【问题案例2】묣은배송 상품 반품 시 배송비 3,000원만 고객에게 청구되는 문제

**场景描述**：위탁판매셀러首次处理반품(退货)，商品为묣은배송(免费配送)。按照规则고객 변심(客户变心)导致的반품应该收取往返6,000원配送费，但판매자센터显示只收取3,000원。

**问题分析**：
- 쿠팡的配送费处理机制与위탁판매자理解的不同
- 묣은배송상품의반품 시 쿠팡会在환불금액中直接扣除3,000원
- 另外3,000원是반품配送费，由客户在반품时单独支付
- 结果客户共支付6,000원，但系统分两次收取，造成误解

**解决方案**：
1. 无需额外向客户要求支付剩余的3,000원
2. 쿠팡会在결제대금환불时自动扣除초도배송비3,000원
3. 반품수거비3,000원由客户在반품접수时支付
4. 판매자无需承担任何配送费损失

**关键要点**：
- 쿠팡의반품배송비처리是自动化的，판매자无需手动计算
- 위탁판매情况下반품수거由판매자自己安排，但배송비척구规则不变
- 有疑问时优先参考쿠팡고객센터공식도움말

## 【问题案例3】AI스마트광고의 무관련 키워드 매칭으로 광고비 낭비

**场景描述**：一位销售구두(皮鞋)的셀러使用AI스마트광고，发现쿠팡AI自动添加了"욕동화(运动鞋)"、"로퍼(乐福鞋)"等与商品不相关的关键词，导致大量无效点击和광고비浪费。

**问题分析**：
- 쿠팡AI광고会自动添加키워드，但这些키워드可能与상품无关
- 调查显示部分셀러因未及时제외(排除)无效키워드，损失了高达1,900만원
- 수동성과형광고除外，其他자동입찰광고都需要单独제외키워드

**解决方案**：
1. 定期检查광고캠페인의자동추가키워드
2. 使用세이비(Savee)等工具进行키워드분석，2周一次효율화
3. 批量제외"돈먹는키워드(烧钱关键词)"
4. 典型案例：구두상품에욕동화키워드제거、셔츠상품에맨투맨키워드제거

**关键要点**：
- AI가추천하는키워드不等于정확한키워드
- 상품과무관련한키워드는즉시제외해야광고낭비방지
- 定期관리시ROAS320%→400%、전환율3.4%→4.2%提升가능

## 【问题案例4】2025년로켓그로스수수료변경으로소형셀러수익악화

**场景描述**：一位月销售额1000万韩元的小卖家，从2025年1月起发现로켓그로스(火箭成长)服务费用大幅增加，原本盈利的产品开始亏损。

**问题分析**：
- 2025년1월6일起로켓그로스수수료구조세분화
- 기존3단계→6단계확대，大部分품목수수료인상
- 반품회수비신설：건당2,100원(20건까지묣은，但로켓그로스반품율고려시무의미)
- 입출고배송요금인상：2,500원~5,000원→3,850원~6,200원

**解决方案**：
1. **选品策略调整**：只选择반품률이극도로낮은필수재(必需品)
2. **品牌建设**：自有브랜드보유시가격핸들링가능
3. **真实利润计算**：반품발생시수수료까지계산한후진짜마진분석
4. **크기조정**：작고가벼운제품중심으로소싱전환

**关键要点**：
- 묣은반품20건제한은로켓그로스반품율을고려하면의미없음
- 무거운상품은비용부담커짐、반품건당추가비용발생
- 소형셀러는반품률낮은상품위주로선별해서리스크감소해야함

## 【问题案例5】아이템위너매칭으로인핸동일상품잘못묶임，노출차단

**场景描述**：셀러的상품突然被아이템위너(商品获胜者)系统묶임(捆绑)，但실제로는전혀다른상품。结果被系统判定为동일상품，메인화면에거의노출되지않음，주문이뚝끊김。

**问题分析**：
- 쿠팡的아이템위너시스템会自动将동일상품으로판단되는상품들을하나의대표상품군으로분류
- 매칭기준：구성、매수、디자인等
- 잘못묶일경우：판매자의기존리뷰까지승자상품에귀속、본인상품은"다른판매자보기"버튼을누르지않으며이동불가

**解决方案**：
1. 쿠팡온라인문의하기
2. 게시글선택："상이한상품과잘못결합되어있으니분리해주세요"
3. 첨부파일양식다운로드및작성(노출ID、분리할옵션ID등)
4. 문의서제출및첨부파일첨부
5. 쿠팡검수후처리완료(상태가'등록'→'해결'로변경)

**预防策略**：
- 상세페이지구성차별화
- 패키징(包装)차별화
- 브랜드상표권등록

## 【问题案例6】판매자점수저하로아이템위너탈락

**场景描述**：셀러原本一直是아이템위너，某天突然탈락。经检查发现판매자점수(卖家分数)从95分下降到82分，导致竞争力下降。

**问题分析**：
- 판매자점수구성요소：
  1. 정시출고완료(按时发货完成)
  2. 정시배송완료(按时配送完成)
  3. 주문이행(订单履行-취소하지않고배송완료한주문)
  4. 24시간내답변(24小时内回复)
- 해당셀러的24시간내답변율이周末공휴일미운영으로인해저하
- 취소율상승(재고관리미흡)

**解决方案**：
1. 고객문의즉시처리：문의는구매의사가높은고객이므로신속응대필수
2. 재고관리강화：품절상품은품절임박상품보다순위하락、품절상태확인은WING수시확인
3. 출고소요기간단축：아이템위너선정시배송속도가동점상품간순위분기요소
4. 정시배송약속준수：약속한빠른배송이실제로지켜지면고객감동

**关键要点**：
- 아이템위너는실시간경쟁，지속적인기준유지필수
- 판매자점수는\[쿠팡윙>판매자정보>판매자점수\]에서확인가능
- 주말、공휴일、판매자휴일은24시간답변계산에서제외
