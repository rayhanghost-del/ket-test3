## 一、Coupang 平台运营核心知识

### 1.1 平台概况
- **成立时间**：2010年
- **市场地位**：韩国电商市场前三，超过2500万用户基础
- **核心优势**："火箭配送"服务（24小时内送达）
- **2024年市场规模**：预计突破164.3万亿韩元

### 1.2 入驻要求与费用

**入驻条件（中国大陆卖家最新要求）**：
- 需提交Amazon、Shopee、eBay等平台最近6个月销售记录证明
- 营业执照等文件需专业翻译并公证认证
- 银行账户户名必须与注册公司法定名称完全一致

**平台费用**：
| 费用类型 | 说明 |
|---------|------|
| 开店费用 | 无保证金、无月租费 |
| 佣金比例 | 按类目5%-10%（3C电子5-8%，服饰10%） |
| 物流费用 | 跨境店可选CGF/CGF Lite服务 |

**结算规则**：
- 结算周期：月度结算，次月第15个工作日支付
- 最低支付金额：当月GMV需超过100万韩元
- 支付货币：韩元(KRW)为主

### 1.3 物流模式

**跨境店物流选择**：
- **初期**：自发货（3-7天妥投）
- **后期**：转CGF Lite提升转化

**本土店物流**：
- 使用火箭仓配送，享次日达服务
- 避免违规索要清关码

| 深圳线 | 深圳市宝安区福永段107国道现代物流园7栋 | 周一、三、五发船（经威海） | 约8天 | **98$/CBM** |

### 1.4 KC 认证要求

**强制认证范围**：
- 婴幼儿及儿童产品
- 电子电器产品
- 照明设备
- 电池等产品

**申请主体限制**：
- 部分KC认证仅能由韩国注册商业实体或分公司申请
- 国际卖家需在韩国设立业务据点或与当地授权进口商合作

**违规处罚**：未能遵守KC认证规定将面临账户冻结等处罚

### 1.5 卖家行为准则（关键要点）

**禁止行为**：
- 创建同一产品多个详情页面
- 操纵销售排名（刷单）
- 订单确定后更改价格
- 人为增加网页流量（机器人/付费点击）
- 损害其他卖家（恶意竞争）
- 付费或提供激励换取评论
- 向客户发送未经请求的营销信息
- 绕过Coupang销售流程（引流到外部网站）

**客户信息保护**：
- 仅能使用客户信息履行订单
- 不得在履行订单范围外联系客户
- 不得与第三方共享客户信息
- 违反《韩国促进信息及通信网络使用和信息保护法》将受惩罚

## 二、韩国电商法律法规

### 2.1 核心法律框架

**《电子商务消费者保护法》**：
- 卖家必须提供有效的联系电话
- 保护消费者知情权和退货权
- 规范电子合同效力

**《外国人投资促进法》**：
- 对外国投资商、外国投资企业提供税收优惠
- 贷款债权人享有特定减免规定

### 2.2 跨境电商进口规定

**中国跨境电商进口税收政策**：
- 单次交易限值：人民币5000元
- 年度交易限值：人民币26000元
- 限值以内：关税税率0%，增值税+消费税按法定税额70%征收

**韩国进口清关**：
- 必须用韩国公司税号清关才能抵扣进口增值税
- "双清包税"渠道无法抵扣，税务风险极高
- 货款和进口清关成本必须保持一致（韩国税务监管严格）

### 2.3 商标与知识产权保护
- 韩国政府高度重视知识产权保护
- 企业应尽早在韩国申请商标保护
- 跨境电商平台提供投诉和维权机制

## 三、韩国税务条款详解

### 3.1 增值税（VAT）

**基本税率**：10%

**征收条件**：
- 年销售额超过3亿韩元（约26万美元）
- 在韩国境内有实体仓库或办事处

**申报时间**：
- 每季度申报（1月、4月、7月、10月）
- 季度结束后25天内完成申报和缴纳
- 外国服务提供商通过税务之家(Hometax)提交申报表

**计算方式**：
```
应纳税额 = 销项税额 - 进项税额
```

**跨境电商VAT征收方式**：
1. **直接征收**：向韩国税务局申请登记，按10%税率征收缴纳
2. **代征**：委托韩国境内支付机构代为征收缴纳

**小规模企业优惠**：
- 年销售额≤8亿韩元可选择简易计税方式
- 年销售额低于3亿韩元可享受减免增值税优惠

### 3.2 企业所得税（法人税）

**阶梯税率**：
| 利润区间 | 含国税 | 地方税 | 合计税率 |
|---------|--------|--------|---------|
| 不足2亿韩元 | 9% | 0.9% | 9.9% |
| 2亿-200亿韩元 | 19% | 1.9% | 20.9% |
| 200亿-3000亿韩元 | 21% | - | 21% |
| 超过3000亿韩元 | 24% | - | 24% |

**申报时间**：
- 每季度预缴（按上一年利润）
- 财年结束后3个月内完成年度申报
- 财政年度12月31日结束的，次年3月底前提交申报

### 3.3 进口关税

**税率类型**：
- **基本税率**：海关法明确规定的基础税率
- **临时税率**：政府根据政策临时调整
- **弹性税率**：反倾销税、反补贴税、季节性税等

**关税范围**：0-30%，根据商品HS编码确定

**进口增值税计算**：
```
进口增值税 = (CIF价格 + 关税) × 10%
```

**关键提示**：
- 进口关税是成本，不可用于后续销售抵扣
- 进口增值税可用于后续销售环节抵扣
- 建议不走包税渠道，否则无法抵扣增值税

### 3.4 个别消费税

**应税商品**：
- 老虎机、博彩用具、猎枪：20%
- 宝石、高级地毯、贵金属、高级包、手表、毛皮、家具：20%
- 石油产品：按升/公斤征收固定税额
- 卷烟：594韩元/20支
- 电子烟：370韩元/毫升

### 3.5 税务合规要点

**三大合规维度**：
1. **进口环节**：关税 + 进口增值税
2. **销售环节**：VAT申报（季度）
3. **企业所得税**：年度申报（阶梯税率）

**合规建议**：
- 聘请专业财务/税务顾问
- 保持准确的财务记录
- 避免逃税、漏税等违规行为
- 持续关注税务法规变化
- 确保货款和清关成本一致

**违规风险**：
- 账户冻结
- 严重法律后果
- 税务稽查趋严（2025年趋势）
