## 二、平台入驻与运营

### 1. Coupang 入驻条件
**企业卖家：**
- 营业执照（有效期至少1个月）
- 法人身份证明（护照或身份证，剩余有效期3个月以上）
- 手持证件自拍照
- 银行账户证明

**入驻流程：**
1. 访问卖家门户网站注册
2. 填写企业信息（英文）
3. 邮箱验证
4. 补充运营信息（产品类目、其他平台经验）
5. 上传文件
6. 绑定收款账户（PingPong或对公账户）
7. 提交审核（4个工作日-1个月）

### 2. 费用结构
- **无店铺租金**
- **佣金**：5%-10%（按品类）
  - 家用电子数码：5%
  - 服装类：10%
  - 化妆品：9%
  - 电子游戏：6%
- **提现费用**：1%-2%

### 3. 产品布局策略
| 产品类型 | 占比 | 用途 |
|----------|------|------|
| 引流款 | 20% | 低价吸引流量 |
| 平价产品 | 30% | 日常销售 |
| 利润款 | 35% | 主要盈利来源 |
| 暴利产品 | 15% | 高毛利单品 |

### 4. 定价策略
- 定价公式：平台佣金10% + 提现费用1-2% + 基础费用 + 基础价 + 利润 + 运费 = 售价
- 定价以 **8或9结尾**，增强价格吸引力
- 确保打8折后仍有 **30%利润**

### 5. 高增长品类
- **消费电子**：年增速15%，智能手机、智能家居设备
- **家居园艺**：收纳用品、小型家电（空气炸锅），中国供应链占比超60%
- **化妆品**：需KC认证
- **服装**：韩流文化带动需求

### 6. 运营要点
- **KC认证**：电子类产品必须通过
- **退换货服务**：退货率低于5%可获得流量倾斜
- **物流时效**：3-7天内妥投最佳
- **本地化支付**：Naver Pay、Kakao Pay覆盖率超90%
- **客服工具**：Naver即时聊天工具

## 三、法律法规合规

### 1. 核心法律框架

| 法律名称 | 主要内容 |
|----------|----------|
| **《电子商务交易消费者保护法》** | 电商运营基础法，规范交易行为 |
| **《个人信息保护法》(PIPA)** | 个人信息收集、使用、存储规范 |
| **《公平标签和广告法》** | 禁止虚假、夸大、欺骗性广告 |
| **《标准合同条例法》** | 确保合同中无不公平条款 |
| **《消费者基本法》** | 消费者纠纷解决和退货政策 |
| **《电子金融交易法》** | 电子支付安全规范 |

### 2. 最新监管动态（2024-2025）

#### 外国平台本地代理人制度
**生效时间**：2024年8月（修订案通过），2026年7月21日正式生效

**触发条件**（满足任一）：
1. 上一年度销售额达到 **1万亿韩元** 以上
2. 月均韩国消费者访问量 **100万以上**
3. 被韩国公平交易委员会要求提交报告

**受影响平台**：AliExpress、Temu等需在韩国设立本地代理人

#### 违规处罚升级（2026年新规）
- 违规统计周期从3年延长至 **5年**
- 重复违规附加金从50%提高至 **100%**
- 主动纠正减免从30%降至 **10%**

### 3. 合规要求清单

**信息披露：**
- 网站首页、产品页、订单确认页必须包含完整披露信息
- 必须提供条款条件和隐私政策链接

**消费者保护：**
- 必须建立消费者投诉处理机制
- 强制提供退换货服务
- 产品安全：删除未获得KC认证产品的销售信息

**个人信息保护：**
- 跨境数据传输必须事先告知用户
- 需对受托方进行隐私保护培训和定期检查
- 数据泄露必须及时通知受影响用户和当局

**广告合规：**
- 禁止虚假、夸大、欺骗性、过度比较性、诽谤性广告
- 未经同意不得发送营销邮件

## 四、税务条款

### 1. 主要税种

#### 增值税（VAT）
- **标准税率**：10%
- **征收对象**：销售商品、提供服务及进口货物的企业
- **跨境电商规定**：
  - **B2C**：外国卖家在韩国有第一笔销售即须注册VAT，负责代收代缴
  - **B2B**：韩国买方按"反向征收"申报缴税

**申报要求：**
- 一般纳税人：月度申报（次月25日前）
- 小规模纳税人（年销售额≤8亿韩元）：可选月度或季度申报
- 通过 **Hometax** 系统申报

#### 企业所得税（法人税）
| 利润区间 | 税率 |
|----------|------|
| ≤2亿韩元 | 9% + 地方税0.9% |
| 2亿～200亿韩元 | 19% + 地方税1.9% |
| 200亿～3000亿韩元 | 21% + 地方税2.1% |
| ＞3000亿韩元 | 24% + 地方税2.4% |

**申报时间：**
- 预缴：季度末次月25日前
- 汇算清缴：次年3月31日前

#### 进口关税与进口增值税
- **进口关税**：根据产品编码税率不同，不可抵扣
- **进口增值税**：10%，可在后续销售环节抵扣
- **税基**：CIF价格（到岸价格）

**重要提示：**
- 韩国进口不走包税，包税无法使用公司抬头，增值税不能抵扣
- 货款和进口清关成本必须保持一致

### 2. 税务合规流程

**销售增值税（季度申报）：**
1. 货物清关（缴纳进口关税和进口增值税）
2. 入仓销售（售价含10%增值税）
3. 季度申报缴纳销售增值税
4. 进口环节缴纳的增值税可用于抵扣

**企业所得税（年度申报）：**
1. 计算营业利润 = 收入 - 货品成本 - 物流成本 - 平台费用 - 广告费用 - 其他杂费
2. 按利润区间计算应纳税额
3. 次年3月31日前申报

### 3. 跨境电商税务要点
- 达到一定销售规模必须注册增值税
- 需准确记录和保存销售、采购、库存等业务数据和凭证
- 建立完善的财务制度，确保数据准确性

## 五、关键要点总结

### 入驻建议
1. **新卖家首选 Coupang**：入驻门槛低、有中文服务、90天免佣期
2. **准备完整材料**：营业执照、法人证件、银行账户
3. **预算规划**：无月租，佣金5-10%，预留30%毛利空间

### 运营策略
1. **品类选择**：消费电子、家居园艺（中国供应链优势）
2. **价格策略**：以8/9结尾，确保折扣后仍有利润
3. **产品布局**：20%引流款 + 35%利润款
4. **物流时效**：3-7天妥投，提供本地退换货

### 合规重点
1. **KC认证**：电子产品必须获得
2. **本地代理人**：如达到一定规模需设立
3. **信息披露**：完整的产品信息和退换货政策
4. **个人信息保护**：跨境传输需用户同意

### 税务规划
1. **VAT注册**：B2C销售需注册，税率10%
2. **进口清关**：使用公司抬头，保留增值税抵扣凭证
3. **账务规范**：准确记录所有业务数据
4. **申报时效**：VAT月度/季度，企业所得税年度
