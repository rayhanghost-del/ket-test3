## 三、基础使用

### 3.1 最简单的使用方式

```bash
python3 coupang_keyword_monitor.py /path/to/你的广告报表.xlsx
```

**示例：**

```bash
python3 coupang_keyword_monitor.py ~/Downloads/广告报表_20260325.xlsx
```

**输出：**
- 报告将生成在当前目录
- 文件名格式：`Coupang关键词监控报告_YYYYMMDD_HHMMSS.xlsx`

### 3.2 指定输出目录

```bash
python3 coupang_keyword_monitor.py 广告报表.xlsx -o ./reports
```

**完整示例：**

```bash
# 创建报告目录
mkdir -p ~/coupang_reports

# 分析报表并输出到指定目录
python3 coupang_keyword_monitor.py \
    ~/Downloads/A01263785_pa_total_campaign_20260301_20260325.xlsx \
    -o ~/coupang_reports

# 查看生成的报告
ls -lh ~/coupang_reports/
```

### 3.3 静默模式（适合定时任务）

```bash
python3 coupang_keyword_monitor.py 广告报表.xlsx -q
```

**特点：**
- 不打印任何信息到控制台
- 只生成Excel报告
- 适合后台自动运行

### 3.4 处理多个报表

**方法1：批量处理脚本**

创建文件 `batch_monitor.sh`：

```bash
#!/bin/bash

REPORT_DIR="~/coupang_reports"
MONITOR_SCRIPT="~/coupang_monitor/coupang_keyword_monitor.py"

# 创建输出目录
mkdir -p $REPORT_DIR

# 处理所有报表
for file in ~/Downloads/*campaign*.xlsx; do
    echo "📂 正在处理: $file"
    python3 $MONITOR_SCRIPT "$file" -o $REPORT_DIR -q
    echo "✅ 完成"
done

echo "📊 所有报告已生成到: $REPORT_DIR"
```

运行：

```bash
chmod +x batch_monitor.sh
./batch_monitor.sh
```

### 4.2 Excel报告详解

生成的Excel包含4个工作表：

#### 工作表1：执行摘要

| 字段 | 说明 |
|------|------|
| 分析时间 | 脚本运行时间 |
| 数据文件 | 源报表文件名 |
| 总数据行数 | 报表总行数 |
| 无效关键词数 | 识别出的问题词数量 |
| 总广告费 | 报表中的总花费 |
| 无效关键词浪费 | 问题词造成的浪费 |
| 浪费占比 | 浪费金额 ÷ 总广告费 |
| 整体ROAS | 总销售额 ÷ 总广告费 |

#### 工作表2：无效关键词清单

包含所有无效关键词的详细数据：

| 列名 | 说明 |
|------|------|
| SKU | 商品名称（简化版） |
| 关键词 | 搜索关键词 |
| 状态 | 问题类型（带emoji） |
| 展示 | 曝光次数 |
| 点击 | 点击次数 |
| 广告费 | 花费金额 |
| 销量 | 成交数量 |
| 销售额 | 成交金额 |
| CTR% | 点击率 |
| ROAS | 投入产出比 |

**使用建议：**
- 按"广告费"列降序排序，优先处理花费高的词
- 按"状态"列筛选，批量处理同类型问题

#### 工作表3：按SKU汇总

统计每个SKU的无效关键词情况：

| 列名 | 说明 |
|------|------|
| SKU | 商品名称 |
| 无效关键词数 | 该SKU的问题词数量 |
| 浪费广告费 | 该SKU的浪费金额 |
| 产生销售 | 该SKU的销售额 |

**使用建议：**
- 找出浪费最严重的SKU
- 针对性地优化这些SKU的关键词

#### 工作表4：TOP50优先处理

最严重的50个关键词，优先停用这些可以节省60-70%的浪费。

### 4.3 根据报告执行优化

**步骤1：打开"TOP50优先处理"工作表**

**步骤2：筛选"高消费零转化"（🚨）**

这些词要立即停用：
- 打开 Coupang Wing
- 进入对应广告活动
- 在"否定关键词"中添加这些词

**步骤3：筛选"有消费无销售"（💸）**

这些词观察1周后再决定：
- 如果1周后仍无转化，加入否定列表
- 如果有转化，保留并观察

**步骤4：筛选"高展示无点击"（⚠️）**

这些词有两种处理方式：
- 降低出价
- 直接停用（如果CTR极低）

## 五、定时自动运行

### 5.1 使用Cron（推荐）

**查看当前cron任务：**

```bash
crontab -l
```

**编辑cron任务：**

```bash
crontab -e
```

**添加定时任务：**

```bash
# 格式: 分 时 日 月 星期 命令

# 示例1: 每周一早上9点运行
0 9 * * 1 /usr/bin/python3 /root/.openclaw/workspace/coupang_keyword_monitor.py /path/to/最新报表.xlsx -o /root/coupang_reports

# 示例2: 每周一和周四早上9点运行
0 9 * * 1,4 /usr/bin/python3 /root/.openclaw/workspace/coupang_keyword_monitor.py /path/to/最新报表.xlsx -o /root/coupang_reports

# 示例3: 每天凌晨3点运行
0 3 * * * /usr/bin/python3 /root/.openclaw/workspace/coupang_keyword_monitor.py /path/to/最新报表.xlsx -o /root/coupang_reports -q

# 示例4: 每月1号运行
0 9 1 * * /usr/bin/python3 /root/.openclaw/workspace/coupang_keyword_monitor.py /path/to/最新报表.xlsx -o /root/coupang_reports
```

**Cron时间格式说明：**

```
┌───────────── 分钟 (0 - 59)
│ ┌───────────── 小时 (0 - 23)
│ │ ┌───────────── 日期 (1 - 31)
│ │ │ ┌───────────── 月份 (1 - 12)
│ │ │ │ ┌───────────── 星期 (0 - 6, 0=周日)
│ │ │ │ │
* * * * * 命令
```

**常用时间示例：**

| 时间设置 | 说明 |
|----------|------|
| `0 9 * * 1` | 每周一9:00 |
| `0 9 * * 1,4` | 每周一和周四9:00 |
| `0 9 * * *` | 每天9:00 |
| `0 */6 * * *` | 每6小时 |
| `0 9 1 * *` | 每月1号9:00 |

### 5.2 使用Systemd定时器（Linux高级用户）

创建服务文件 `/etc/systemd/system/coupang-monitor.service`：

```ini
[Unit]
Description=Coupang Keyword Monitor
After=network.target

[Service]
Type=oneshot
ExecStart=/usr/bin/python3 /root/.openclaw/workspace/coupang_keyword_monitor.py /path/to/报表.xlsx -o /root/coupang_reports
User=root
```

创建定时器文件 `/etc/systemd/system/coupang-monitor.timer`：

```ini
[Unit]
Description=Run Coupang Keyword Monitor weekly

[Timer]
OnCalendar=Mon *-*-* 09:00:00
Persistent=true

[Install]
WantedBy=timers.target
```

启用定时器：

```bash
systemctl daemon-reload
systemctl enable coupang-monitor.timer
systemctl start coupang-monitor.timer

# 查看状态
systemctl list-timers --all
```

### Q3: 报表列名不匹配

**症状：** 脚本运行但结果为空或报错

**解决：**

检查报表是否包含以下列名之一：
- 关键词 / 키워드
- 广告费 / 광고비
- 展示次数 / 노출수
- 点击次数 / 클릭수
- 总销量（14天） / 총판매량(14일)
- 总转化销售额（14天） / 총전환매출(14일)
- 广告商品名 / 광고상품명

创建HTML模板功能：

```python
def export_html_report(self, filename=None):
    """导出HTML报告"""
    if filename is None:
        timestamp = datetime.now().strftime('%Y%m%d_%H%M%S')
        filename = f'Coupang关键词监控报告_{timestamp}.html'
    
    output_path = os.path.join(self.output_dir, filename)
    
    html_template = '''
    <!DOCTYPE html>
    <html>
    <head>

        <title>Coupang关键词监控报告</title>

            th { background: #4CAF50; color: white; }
            tr:nth-child(even) { background: #f2f2f2; }
            .warning { color: #ff9800; }
            .danger { color: #f44336; }
            .success { color: #4CAF50; }
        </style>
    </head>
    <body>
        <h1>📊 Coupang关键词监控报告</h1>
        <div class="summary">
            <h2>执行摘要</h2>
            <p><strong>分析时间:</strong> {分析时间}</p>
            <p><strong>数据文件:</strong> {数据文件}</p>

            <p><strong>无效关键词数:</strong> <span class="danger">{无效关键词数}</span></p>
            <p><strong>浪费金额:</strong> <span class="danger">₩{无效关键词浪费:,.0f}</span></p>
            <p><strong>浪费占比:</strong> <span class="{waste_class}">{浪费占比}</span></p>
        </div>
        
        <h2>无效关键词清单</h2>
        <table>
            <tr>
                <th>SKU</th>
                <th>关键词</th>
                <th>状态</th>
                <th>广告费</th>
                <th>销售额</th>
            </tr>
            {keyword_rows}
        </table>
    </body>
    </html>
    '''
    
    # 生成表格行
    rows = ""
    for _, row in self.invalid_keywords.head(100).iterrows():
        rows += f"<tr><td>{row['广告商品名'][:30]}</td><td>{row['关键词']}</td><td>{row['关键词状态']}</td><td>₩{row['广告费']:,.0f}</td><td>₩{row['总转化销售额（14天）']:,.0f}</td></tr>\n"
    
    # 确定样式类
    roas_class = "success" if self.summary['整体ROAS'] >= 4 else "warning" if self.summary['整体ROAS'] >= 2 else "danger"
    waste_class = "success" if float(self.summary['浪费占比'].rstrip('%')) < 15 else "warning" if float(self.summary['浪费占比'].rstrip('%')) < 25 else "danger"
    
    # 填充模板
    html_content = html_template.format(
        **self.summary,
        keyword_rows=rows,
        roas_class=roas_class,
        waste_class=waste_class
    )
    
    with open(output_path, 'w', encoding='utf-8') as f:
        f.write(html_content)
    
    print(f"✅ HTML报告已生成: {output_path}")
    return output_path
```

import sqlite3

def save_to_database(self, db_path='coupang_monitor.db'):
    """保存到SQLite数据库"""
    conn = sqlite3.connect(db_path)
    cursor = conn.cursor()
    
    # 创建表
    cursor.execute('''
        CREATE TABLE IF NOT EXISTS keyword_reports (
            id INTEGER PRIMARY KEY AUTOINCREMENT,
            analysis_time TEXT,
            file_name TEXT,
            total_cost REAL,
            total_sales REAL,
            roas REAL,
            invalid_keywords_count INTEGER,
            invalid_cost REAL,
            waste_ratio TEXT
        )
    ''')
    
    # 插入数据
    cursor.execute('''
        INSERT INTO keyword_reports
