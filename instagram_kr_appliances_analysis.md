### 6.1 韩系家居风格核心特征

基于对韩系Instagram家居账号的深度分析，总结出以下核心特征：

#### 色彩体系
```
主色调（60%）：纯白色、米白色、浅灰色
辅助色（25%）：原木色、浅木色、绿植绿色
点缀色（15%）：黑色、金属色、莫兰迪色系
```

#### 空间特点
1. **"有组织的杂乱"** - 物品丰富但摆放有序，呈现生活气息
2. **白色基调** - 墙面、地板、天花板作为画布，突出家具
3. **单品精选** - 每件家具/电器都能独立成为视觉焦点
4. **自然光线** - 大窗户引入充足自然光，营造明亮氛围

#### 标志性元素
- Samsung Serif TV / Frame TV
- Louis Poulsen 灯具 (PH5, Panthella)
- Vitra Eames 家具
- 龟背竹、琴叶榕等绿植

## 第七部分：关键洞察与建议

### 6.1 内容创作建议

**吸尘器内容：**
1. ✅ 强调与浅色家居的融合
2. ✅ 展示收纳/充电座的整洁感
3. ✅ 前后对比（清洁效果）
4. ✅ 自然光下的真实使用场景
5. ❌ 避免过于工业化的展示

**吹风机内容：**
1. ✅ 梳妆台场景比浴室更受欢迎
2. ✅ 粉色/白色产品更易获得互动
3. ✅ 搭配其他护发产品增加丰富度
4. ✅ 展示风力效果（头发飘动）
5. ❌ 避免纯产品图，需要场景支撑

### 6.2 视觉风格建议

| 元素 | 推荐方案 |
|------|----------|
| 光线 | 自然光 > 暖光 > 冷光 |
| 背景 | 纯色/极简 > 复杂场景 |
| 配色 | 白+木+绿 > 多彩搭配 |
| 构图 | 留白 > 填满 |
| 氛围 | 治愈 > 功能 |

### 6.3 平台算法友好特征

1. **首图吸引力**：产品+场景结合
2. **色彩统一性**：单图或九宫格色调一致
3. **生活方式感**：不只是产品，是生活
4. **真实感**：避免过度修图
5. **互动引导**：提问式caption
