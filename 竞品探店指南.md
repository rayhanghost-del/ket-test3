# 英语培训机构探店指南（家长身份）

## 一、探店前准备

### 身份设定
- **角色**：家长，孩子三年级（或根据实际调整）
- **需求**：想给孩子找英语培训班，了解附近的选择
- **预算**：不要主动透露，让对方先报价

### 携带物品
- 手机（录音/拍照，注意别被发现）
- 笔记本（记录关键信息）
- 孩子照片（可选，增加可信度）

---

## 二、探店流程与话术

### 阶段1：进门观察（5分钟）

#### 看什么
| 观察项 | 具体内容 | 为什么重要 |
|--------|----------|------------|
| **门头/招牌** | 品牌名、LOGO、是否亮灯 | 品牌形象和营业状态 |
| **前台接待** | 几个人、是否专业、态度如何 | 服务标准 |
| **大厅环境** | 装修档次、整洁度、家长休息区 | 定位高端/平价 |
| **展示墙** | 学员作品、获奖证书、师资介绍 | 教学成果展示 |
| **在读学员** | 大概人数、年龄段分布 | 真实生源情况 |

#### 记什么
```
时间：____年____月____日 ____点
地址：___________________________
门头照片：□已拍 □未拍
第一印象：□高端 □中等 □普通
客流量：□多 □少 □无其他家长
```

---

### 阶段2：咨询对话（15-20分钟）

#### 开场白
> "你好，我孩子现在三年级，想找个英语班。听说你们在这边口碑不错，过来了解一下。"

#### 必问问题清单

##### 【课程类】
| 问题 | 目的 | 对方可能的回答 |
|------|------|----------------|
| "你们这边针对三年级的有什么课程？" | 了解课程体系 | 记录课程名称、内容、目标 |
| "一周几次课？一次多长时间？" | 了解课时安排 | 计算月课时量 |
| "一个班几个孩子？" | 了解班型 | 小班6-10人/中班12-15人/大班 |
| "用什么教材？是原版还是自编？" | 了解教学内容 | 剑桥/牛津/朗文/自编 |
| "有外教吗？外教是哪国的？" | 了解师资配置 | 欧美/菲律宾/中教 |

##### 【价格类】⭐ 核心
| 问题 | 话术技巧 | 记录要点 |
|------|----------|----------|
| "学费怎么收？" | 让对方先报整体价格 | 年费/学期费/课时包 |
| "如果只报一个学期多少钱？" | 测试灵活性 | 是否有短周期课程 |
| "现在报名有什么优惠吗？" | 了解促销策略 | 折扣/赠课/礼品 |
| "退费政策是怎样的？" | 了解风险 | 能否退费/退费比例/时限 |
| "除了学费还有其他费用吗？" | 问清隐形消费 | 教材费/注册费/活动费 |

##### 【师资类】
| 问题 | 目的 |
|------|------|
| "任课老师是什么背景？" | 了解老师资质 |
| "老师是固定的还是经常换？" | 了解稳定性 |
| "可以试听吗？试听收费吗？" | 了解获客策略 |

##### 【竞争类】（侧面打听）
| 问题 | 话术 |
|------|------|
| "附近好像还有其他英语机构，你们和它们有什么区别？" | 让对方主动说竞品坏话 |
| "你们这边开了多久了？" | 了解经营稳定性 |
| "在这上课的孩子多吗？" | 验证人气 |

---

### 阶段3：要求参观（10分钟）

#### 说什么
> "我能看看教室吗？想了解一下上课环境。"

#### 看什么
| 区域 | 观察重点 |
|------|----------|
| **教室** | 大小、采光、桌椅数量、教学设备（投影/白板/电视） |
| **卫生状况** | 地面、桌面、垃圾桶是否干净 |
| **安全设施** | 消防通道、监控、门窗防护 |
| **其他教室** | 是否有空教室（利用率） |
| **卫生间** | 是否干净（细节体现管理） |

#### 记什么
```
教室数量：____间
每间容量：____人
设备情况：□投影 □白板 □电视 □其他____
卫生评分：□很好 □一般 □较差
安全隐患：________________
```

---

### 阶段4：试听/体验（如有）

#### 如果对方邀请试听
- 接受，但说"今天时间紧，改天带孩子来"
- 目的是了解试听课流程和转化话术

#### 问什么
| 问题 | 目的 |
|------|------|
| "试听课是什么样的内容？" | 了解试听课设计 |
| "试听后怎么知道孩子适不适合？" | 了解评估方式 |
| "试听当天报名有优惠吗？" | 了解转化策略 |

---

### 阶段5：离开前（5分钟）

#### 说什么
> "我回去和孩子爸爸/妈妈商量一下，能给我一份资料吗？"

#### 要什么
- [ ] 课程介绍单页/手册
- [ ] 价目表（如有）
- [ ] 老师名片/联系方式
- [ ] 试听预约方式

#### 最后试探
> "如果我下周带孩子来试听，现在能预约吗？"

## 五、探店注意事项

### DO（要做）
- ✅ 以家长身份自然询问
- ✅ 多问开放式问题（让销售多说）
- ✅ 记录关键数字（价格、人数、课时）
- ✅ 观察细节（卫生、设备、人员状态）
- ✅ 拿资料（回头分析）

### DON'T（不要做）
- ❌ 不要透露自己是同行
- ❌ 不要当场表现出太贵/太便宜的情绪
- ❌ 不要拍照太明显（可以假装回消息时拍）
- ❌ 不要待太久（30-40分钟足够）
- ❌ 不要当天就做决定（说"回去商量"）
