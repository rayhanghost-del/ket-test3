## 二、韩系原生风格特征总结

### 2.1 色彩体系

**主色调:**
- **白色 (White)** - 墙面、大件家具 (占比 45%)
  - 色号参考: #F5F5F5, #FAFAFA, #FFFFFF
  - 不是纯白，而是带微暖的米白/奶白
  
- **原木色 (Wood)** - 地板、家具、装饰 (占比 35%)
  - 橡木色: #D4A574, #C4956A
  - 胡桃木: #8B6914, #A0826D
  - 偏好浅色系原木，避免深红/深棕

- **绿色点缀 (Green Accent)** - 植物、软装 (占比 15%)
  - 莫兰迪绿: #8FBC8F, #90EE90
  - 鼠尾草绿: #9DC183, #8F9779
  - 避免荧光绿、过于鲜艳的绿

- **中性色 (Neutral)** - 灰色、米色 (占比 5%)
  - 浅灰: #E8E8E8, #D3D3D3
  - 米色: #F5F5DC, #FFF8DC

**配色比例:**
```
60% 白色/米白 (基础)
30% 原木色 (家具/地板)
7%  绿色/植物 (点缀)
3%  其他中性色
```

### 2.2 空间布局特点

**小户型优化 (韩国特色):**
- 平均居住面积: 60-85平米 (公寓)
- 开放式厨房+客厅 (오픈형 거실)
- 多功能空间设计 (书房兼客房)
- 垂直收纳最大化

**典型布局元素:**
1. **玄关 (현관)**
   - 下沉式或视觉分区
   - 鞋柜+穿衣镜+小挂钩
   - 尺寸: 1.5-2平米

2. **客厅 (거실)**
   - 沙发不靠墙 (岛式摆放)
   - 低矮家具为主
   - 电视墙简约，隐藏式收纳

3. **卧室 (침실)**
   - 地台床或矮床架
   - 床头不靠墙 (留出通道)
   - 嵌入式衣柜

4. **厨房 (주방)**
   - 一字型或L型布局
   - 白色/浅色橱柜
   - 嵌入式电器

### 2.3 家具风格

**必备单品:**
- **矮桌/茶几** - 原木材质，高度 30-40cm
- **懒人沙发** - 豆袋、蒲团式
- **开放式书架** - 原木或白色金属
- **藤编/草编收纳** - 自然材质
- **落地灯** - 纸罩、木质灯杆

**材质偏好:**
1. 原木 > 板材 (可见木纹)
2. 棉麻 > 化纤 (自然质感)
3. 藤编 > 塑料 (手工感)
4. 陶瓷 > 金属 (温润感)

**避免元素:**
- 欧式雕花家具
- 皮质大沙发
- 金属玻璃茶几
- 过于现代的塑料家具

### 2.4 装饰元素

**植物选择:**
- 龟背竹 (Monstera) - 出镜率最高
- 琴叶榕 (Fiddle Leaf Fig)
- 虎尾兰 (Snake Plant)
- 绿萝 (Pothos) - 垂吊
- 多肉植物组合

**软装织物:**
- 亚麻窗帘 (米色/白色)
- 针织毯子 (大地色系)
- 棉麻抱枕 (素色/细条纹)
- 草编地毯/地垫

**墙面装饰:**
- 留白为主 (60%以上)
- 单幅大画 > 照片墙
- 木质搁板展示
- 小圆镜组合

**灯光氛围:**
- 色温: 2700K-3000K (暖黄)
- 多层次照明: 主灯+台灯+落地灯
- 隐藏式灯带 (间接照明)
- 避免冷白光

### 2.5 产品图风格 (Coupang分析)

**拍摄特点:**
1. **场景化展示**
   - 产品置于真实家居环境
   - 不是纯白背景棚拍
   - 展示产品在实际使用中的状态

2. **自然光线**
   - 白天自然光拍摄
   - 避免闪光灯硬光
   - 柔和阴影，真实质感

3. **生活化细节**
   - 产品周围有真实使用痕迹
   - 搭配其他家居物品
   - 展示尺寸比例 (放人手/常见物品对比)

4. **配色协调**
   - 产品图背景与韩系配色一致
   - 白+木+绿背景最常见
   - 避免杂乱背景

**各品类特点:**

**무선청소기 (无线吸尘器):**
- 展示收纳充电座
- 强调轻便设计 (女性手持)
- 场景: 木地板、地毯、沙发缝隙
- 配色: 白/银/浅灰为主

**헤어드라이기 (吹风机):**
- 浴室场景拍摄
- 强调快速干发效果
- 展示便携性
- 配色: 白/粉/薄荷绿

**요가매트 (瑜伽垫):**
- 客厅/卧室场景
- 展示卷起收纳状态
- 强调厚度/防滑
- 配色: 莫兰迪色系 (灰粉/灰绿/米色)

## 三、与国内外风格对比

### 3.1 韩系 vs 国内网红风格

| 维度 | 韩系原生风格 | 国内网红风格 |
|------|--------------|--------------|
| **色彩** | 白+木+绿，低饱和 | 大理石+金，高对比 |
| **材质** | 原木、棉麻、藤编 | 大理石、金属、玻璃 |
| **家具** | 低矮、简约、实用 | 高大、装饰性强 |
| **植物** | 真实绿植，自然状态 | 仿真花/过度修剪 |
| **灯光** | 暖黄光，多层次 | 冷白光，主灯为主 |
| **收纳** | 隐藏式，整洁 | 开放式展示 |
| **氛围** | 温馨、生活化 | 精致、样板间感 |
| **拍照** | 自然光，真实场景 | 滤镜重，摆拍感 |

**典型差异示例:**
- 韩系: 原木茶几上放着正在看的书和喝了一半的茶
- 国内网红: 大理石茶几上精心摆放的ins风道具组合

### 3.2 韩系 vs 欧美极简风格

| 维度 | 韩系原生风格 | 欧美极简风格 |
|------|--------------|--------------|
| **色彩** | 暖白+原木+绿 | 黑白灰为主 |
| **温度感** | 温暖、有人情味 | 冷静、疏离感 |
| **材质** | 自然材质为主 | 工业材质为主 |
| **装饰** | 适度装饰，生活痕迹 | 极少装饰，空无一物 |
| **植物** | 大型绿植，自然生长 | 小型多肉，几何造型 |
| **家具** | 圆润线条，舒适优先 | 直线条，形式优先 |
| **空间** | 紧凑但温馨 | 开阔但冷清 |

**典型差异示例:**
- 韩系: 白色墙面+原木家具+大型龟背竹+针织毯
- 欧美极简: 水泥墙面+黑色金属家具+单株仙人掌+无纺织品

### 3.3 韩系 vs 日系风格

| 维度 | 韩系原生风格 | 日系风格 |
|------|--------------|----------|
| **色彩** | 白+木+绿 (清新) | 原木+白+米 (温润) |
| **饱和度** | 略高，更明亮 | 更低，更沉稳 |
| **家具** | 现代简约+北欧元素 | 传统日式+现代融合 |
| **收纳** | 隐藏式，整洁 | 展示式，有序 |
| **植物** | 大型绿植为主 | 小型盆栽为主 |
| **氛围** | 年轻、活力 | 禅意、宁静 |
| **装饰** | 适度装饰品 | 极少装饰 |

**核心区别:**
- 韩系更明亮、更现代、更"ins风"
- 日系更沉稳、更传统、更"侘寂"

## 七、核心结论

### 7.1 韩系原生风格核心特征

**一句话总结:**
> 韩系原生风格是"温暖的白"——以白色和原木为基底，用绿色点缀，营造温馨、自然、生活化的居住氛围。

**三大支柱:**
1. **色彩: 白+木+绿** - 低饱和、高温暖度
2. **材质: 自然优先** - 原木、棉麻、藤编
3. **氛围: 生活化** - 真实使用痕迹，非样板间

**与国内的本质区别:**
- 韩系追求"温馨的生活感"
- 国内网红追求"精致的展示感"

### 7.2 实用建议

**内容创作:**
- 使用暖色调白平衡 (3000K左右)
- 选择自然光拍摄时段
- 保留适度的生活痕迹
- 优先展示韩国本土品牌

**产品选择:**
- 家具: 低矮、原木、圆润线条
- 植物: 大型绿植，自然生长状态
- 织物: 棉麻材质，大地色系
- 灯光: 暖黄光，多层次照明

**避免踩坑:**
- 不要使用冷白光
- 不要过度摆拍
- 不要堆砌装饰品
- 不要使用中文水印内容
